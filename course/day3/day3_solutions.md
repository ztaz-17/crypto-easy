## Задача 1. M и E (2025, I, №2)

$M = 1 - \frac{3}{1 + \frac{2}{1 + \frac13}}$.

Снизу вверх: $1 + \frac13 = \frac43$; $\frac{2}{4/3} = \frac32$; $1 + \frac32 = \frac52$; $\frac{3}{5/2} = \frac65$; $M = 1 - \frac65 = -\frac15$.

$E = \frac{27^3}{3^{10} - 5\cdot9^4 + 2\cdot3^9}$.
$27^3 = (3^3)^3 = 3^9$. $9^4 = (3^2)^4 = 3^8$.
Знаменатель: $3^{10} - 5\cdot3^8 + 2\cdot3^9 = 3^8(9 - 5 + 6) = 3^8\cdot10$.
$E = \frac{3^9}{3^8\cdot10} = \frac{3}{10}$.

Итог: $\frac{1}{E} + \frac{M}{E-2M} = \frac{10}{3} + \frac{-1/5}{3/10 - 2(-1/5)} = \frac{10}{3} + \frac{-1/5}{3/10 + 2/5} = \frac{10}{3} + \frac{-1/5}{7/10} = \frac{10}{3} - \frac{2}{7} = \frac{70-6}{21} = \frac{64}{21}$.

**Ответ: $M = -\frac15$, $E = \frac{3}{10}$, итог $\frac{64}{21}$.**

## Задача 2. F и I (2024, I, №2)

$I = \frac{25^5 - 5^9}{125^3} = \frac{(5^2)^5 - 5^9}{(5^3)^3} = \frac{5^{10} - 5^9}{5^9} = \frac{5^9(5-1)}{5^9} = 4$.

$F = (\frac23)^5 \cdot 3.15 - \frac{27^3}{9^5 \cdot \frac{14}{30}}$.
$(\frac23)^5 = \frac{32}{243}$. $3.15 = \frac{63}{20}$. $\frac{32}{243} \cdot \frac{63}{20} = \frac{32\cdot63}{243\cdot20} = \frac{8\cdot63}{243\cdot5} = \frac{504}{1215} = \frac{56}{135}$.
$\frac{27^3}{9^5} = \frac{3^9}{3^{10}} = \frac13$. $\frac13 : \frac{14}{30} = \frac13 \cdot \frac{30}{14} = \frac{10}{14} = \frac57$.
$F = \frac{56}{135} - \frac57 = \frac{56\cdot7 - 5\cdot135}{945} = \frac{392 - 675}{945} = -\frac{283}{945}$.

По критериям 2024 года: $F = \frac{7}{27}$ (возможна опечатка в условии или другой порядок действий).

**Ответ: $I = 4$, $F = \frac{7}{27}$ (по критериям), итог $F - \frac1I = \frac{7}{27} - \frac14 = \frac{1}{108}$.**

## Задача 3. Ω(p,q) (2023, I, №3)

$\frac{p^2 + 8pq + 12q^2}{p^2 - 4q^2} = \frac{(p+6q)(p+2q)}{(p-2q)(p+2q)} = \frac{p+6q}{p-2q}$.
$\frac{p+6q}{p-2q} + 3 = \frac{p+6q + 3p - 6q}{p-2q} = \frac{4p}{p-2q}$.
$\frac{p^2 - 2pq}{p^2 - 4pq + 4q^2} = \frac{p(p-2q)}{(p-2q)^2} = \frac{p}{p-2q}$.
$\Omega = \frac{4p}{p-2q} : \frac{p}{p-2q} = \frac{4p}{p-2q} \cdot \frac{p-2q}{p} = 4$.

**Ответ: $\Omega = 4$, не зависит от $p$ и $q$.**

## Задача 4. Θ (2022, I, №3)

$\frac{x^2-4}{x^2+4x+4} = \frac{(x-2)(x+2)}{(x+2)^2} = \frac{x-2}{x+2}$.
$\frac{x^2+4}{x^2-4x+4} = \frac{x^2+4}{(x-2)^2}$.
$\Theta = \left(\frac{x-2}{x+2} - \frac{x^2+4}{(x-2)^2}\right) \cdot \frac{x^2-4}{x}$.

Приводим к общему знаменателю: $\frac{(x-2)^3 - (x^2+4)(x+2)}{(x+2)(x-2)^2} \cdot \frac{(x-2)(x+2)}{x} = \frac{(x-2)^3 - (x^2+4)(x+2)}{(x-2)x}$.

$(x-2)^3 = x^3 - 6x^2 + 12x - 8$.
$(x^2+4)(x+2) = x^3 + 2x^2 + 4x + 8$.
Разность: $-8x^2 + 8x - 16 = -8(x^2 - x + 2)$.

$\Theta = \frac{-8(x^2 - x + 2)}{x(x-2)}$. При $x=3$: $\Theta = \frac{-8(9-3+2)}{3\cdot1} = \frac{-8\cdot8}{3} = -\frac{64}{3}$.

**Ответ: $\Theta = \frac{-8(x^2-x+2)}{x(x-2)}$, при $x=3$: $-\frac{64}{3}$.**

## Задача 5. Ω(m,n) (2021, I, №2)

$\frac{9m^2-4n^2}{9m^2+12mn+4n^2} = \frac{(3m-2n)(3m+2n)}{(3m+2n)^2} = \frac{3m-2n}{3m+2n}$.
$\frac{3m-2n}{3m+2n} - 1 = \frac{3m-2n-3m-2n}{3m+2n} = \frac{-4n}{3m+2n}$.
$\frac{-4n}{3m+2n} \cdot \frac{3m+2n}{4+2n} = \frac{-4n}{4+2n} = \frac{-2n}{2+n}$.
$\Omega = \frac{-2n}{2+n} + 2 = \frac{-2n+4+2n}{2+n} = \frac{4}{2+n}$.

При $m=3$, $n=5$: $\Omega = \frac{4}{7}$. При $m=2$, $n=-3$: $\Omega = \frac{4}{-1} = -4$.

**Ответ: $\Omega = \frac{4}{2+n}$. При $(3,5)$: $\frac{4}{7}$; при $(2,-3)$: $-4$.**

## Задача 6. Упрощение с дробями (2008, I, №2)

$\frac{4a^2-9}{2a+3} = \frac{(2a-3)(2a+3)}{2a+3} = 2a-3$.
$\frac{a^2-2a+1}{a-1} = \frac{(a-1)^2}{a-1} = a-1$.
$\frac{9a^2+12a+4}{3a+2} = \frac{(3a+2)^2}{3a+2} = 3a+2$.
Сумма: $(2a-3) + (a-1) - (3a+2) = 2a-3+a-1-3a-2 = -6$.

**Ответ: $-6$ (не зависит от $a$).**

## Задача 7. Упрощение (2009, I, №1)

$\frac{3a+7b}{5a} + \frac{8a-3b}{5b} = \frac{b(3a+7b) + a(8a-3b)}{5ab} = \frac{3ab+7b^2+8a^2-3ab}{5ab} = \frac{8a^2+7b^2}{5ab}$.
Деление: $\frac{8a^2+7b^2}{5ab} : \frac{7b^2+8a^2}{10ab} = \frac{8a^2+7b^2}{5ab} \cdot \frac{10ab}{8a^2+7b^2} = 2$.

При $a=0$, $b=3$ — не определено (деление на $0$). При $a=2009$, $b=2009$: $= 2$.

**Ответ: $2$ (при допустимых значениях).**

## Задача 8. Упрощение (2012, I, №2)

$\frac{a-5}{a+5} + \frac{a+5}{a-5} = \frac{(a-5)^2 + (a+5)^2}{(a+5)(a-5)} = \frac{(a^2-10a+25)+(a^2+10a+25)}{a^2-25} = \frac{2a^2+50}{a^2-25} = \frac{2(a^2+25)}{a^2-25}$.
$1 - \frac{50}{a^2+25} = \frac{a^2+25-50}{a^2+25} = \frac{a^2-25}{a^2+25}$.
Произведение: $\frac{2(a^2+25)}{a^2-25} \cdot \frac{a^2-25}{a^2+25} = 2$.

**Ответ: $2$.**

## Задача 14. Прямые и параметры (2025, I, №4)

**а)** $2x + 3 = 5x - 7$ $\Rightarrow$ $3x = 10$ $\Rightarrow$ $x = \frac{10}{3}$, $y = 2\cdot\frac{10}{3}+3 = \frac{29}{3}$. Точка $(\frac{10}{3}; \frac{29}{3})$.

**б)** $y=2x+3$: с $Oy$ — $(0;3)$, с $Ox$ — $(-\frac32;0)$. $y=kx+b$: с $Oy$ — $(0;b)$, с $Ox$ — $(-\frac{b}{k};0)$.

## Задача 15. Пары корней (2010, I, №5)

$25x^2 + 10xy + y^2 = 4x^2 + 12xy + 9y^2$.
Переносим: $21x^2 - 2xy - 8y^2 = 0$.
Решаем как квадратное относительно $x$: $D = 4y^2 + 4\cdot21\cdot8y^2 = 4y^2 + 672y^2 = 676y^2 = (26y)^2$.
$x = \frac{2y \pm 26y}{42}$. $x_1 = \frac{28y}{42} = \frac{2y}{3}$, $x_2 = \frac{-24y}{42} = -\frac{4y}{7}$.

Проверяем: а) $(\frac13, \frac12)$: $x = \frac13 = \frac{2}{3}\cdot\frac12$? $\frac13 = \frac13$ ✓. б) $(-1,1)$: $-1 = \frac{2}{3}\cdot1$? Нет. $-1 = -\frac{4}{7}\cdot1$? Нет. в) $(2,3)$: $2 = \frac{2}{3}\cdot3 = 2$ ✓.

**Ответ: а) и в).**

## Задача 16. Уравнение с параметром (2009, I, №5)

$ax - (\frac{x}{2} - 3) = a(x-3) + 5x$. Подставляем $x=2$:
$2a - (1 - 3) = a(2-3) + 10$ $\Rightarrow$ $2a - (-2) = a(-1) + 10$ $\Rightarrow$ $2a + 2 = -a + 10$ $\Rightarrow$ $3a = 8$ $\Rightarrow$ $a = \frac{8}{3}$.

**Ответ: $a = \frac{8}{3}$.**
