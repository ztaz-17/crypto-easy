## Задача 1. Четырёхугольник (2024, I, №6)

**а) ∠D = 60°**

В $\triangle ABC$: $AB = BC$, $\angle B = 120°$. $\angle BAC = \angle BCA = (180°-120°):2 = 30°$ (свойство равнобедренного треугольника: углы при основании равны).

$\angle BCA = \angle CAD = 30°$ — это накрест лежащие углы при прямых $BC$ и $AD$ и секущей $AC$. Значит, $BC \parallel AD$, и $ABCD$ — трапеция.

$\angle ACD = \angle BCD - \angle BCA = 120° - 30° = 90°$.
В $\triangle ACD$: $\angle ACD = 90°$, $\angle CAD = 30°$. По сумме углов треугольника: $\angle D = 180° - 90° - 30° = 60°$.

**б) BD = 7**

В $\triangle ACD$: $\angle ACD = 90°$, $\angle CAD = 30°$ $\Rightarrow$ $\angle D = 60°$.
По свойству прямоугольного треугольника (катет против угла 30° равен половине гипотенузы): $CD = \frac{AD}{2}$.

$\triangle BCD$ — равнобедренный ($BC = CD$). $\angle BCD = 120°$ $\Rightarrow$ $\angle CBD = \angle BDC = (180°-120°):2 = 30°$.

Проведём высоту $CF \perp BD$ ($F$ — на $BD$). В $\triangle BCF$ ($\angle BFC = 90°$):
$\angle CBF = 30°$ $\Rightarrow$ по свойству (катет против 30°) $CF = \frac{BC}{2}$.
В $\triangle ACD$: $AC = 7$, $\angle CAD = 30°$, $\angle ACD = 90°$.
По свойству (катет против 30°): $CD = \frac{AD}{2}$.
$\triangle AHC$ ($H$ — из части в): $\angle CAH = 30°$ $\Rightarrow$ $CH = \frac{AC}{2} = 3,5$.
В $\triangle CHD$: $\angle CDH = 60°$ $\Rightarrow$ $\angle DCH = 30°$.
$HD = \frac{CD}{2}$ (катет против 30°).

Из $\triangle AHC$ ($\angle AHC = 90°$, $\angle CAH = 30°$): $AH = \frac{AC}{\sqrt{3}}\cdot\sqrt{3}=...$

**Упрощённое решение для BD:**
$\triangle ACD$ — прямоугольный с углом 30°.
$\triangle BCD$ — равнобедренный с $\angle BCD = 120°$, $\angle CBD = \angle BDC = 30°$.
В $\triangle ACD$: $CD = \frac{AD}{2}$ (свойство 30°).
Достроим $\triangle ACD$ до равностороннего: продолжим $CD$ за $D$ на $CD$ до точки $C'$ так, что $C'D = CD$.
Тогда $CC' = 2CD = AD$. $\triangle ACC'$ — равнобедренный. $\angle ACD = 90°$, $\angle CDC' = 180° - \angle ADC = 180° - 60° = 120°$, $\angle DCC' = \angle DC'C = 30°$.
$\angle ACC' = 90° + 30° = 120°$, $\angle AC'C = 30° + 30° = 60°$.
В $\triangle ACC'$: сумма углов $120° + \angle C'AC + 60° = 180°$ $\Rightarrow$ $\angle C'AC = 0°$... 

*Другой способ:* Поскольку $\angle CBD = \angle BDC = 30°$, $\triangle BCD$ — равнобедренный ($BC = CD$).
Проведём $CM \perp BD$ (точка $M$ — середина $BD$, т.к. в равнобедренном треугольнике высота является медианой).
В $\triangle BCM$: $\angle CBM = 30°$, $\angle BMC = 90°$ $\Rightarrow$ $BM = \frac{BC\sqrt{3}}{2}$.
$BC = CD$ из $\triangle ACD$: $CD = \frac{AD}{2}$, $AC = 7$. В прямоугольном $\triangle ACD$ со стороной $AC = 7$ и углом $30°$:
$\frac{AC}{CD} = \frac{7}{CD} = \sqrt{3}$ (соотношение катетов в треугольнике 30°-60°-90°: катет против 60° в √3 раз больше катета против 30°).
$CD = \frac{7}{\sqrt{3}}$, $BC = \frac{7}{\sqrt{3}}$.
$BM = \frac{\frac{7}{\sqrt{3}} \cdot \sqrt{3}}{2} = \frac{7}{2}$.
$BD = 2 \cdot BM = 2 \cdot \frac{7}{2} = 7$.

**в) CH = 3,5**

$CH \perp AD$, $\angle CAD = 30°$. В $\triangle ACH$: $\angle AHC = 90°$, $\angle CAH = 30°$.
По свойству прямоугольного треугольника (катет против угла 30°): $CH = \frac{AC}{2} = \frac{7}{2} = 3,5$.

## Задача 2. Прямоугольный треугольник (2025, I, №6)

**а) AL = 10**

В $\triangle ABC$: $\angle C = 90°$, $\angle A = 30°$ $\Rightarrow$ $\angle B = 60°$.
$BL$ — биссектриса $\angle B = 60°$ $\Rightarrow$ $\angle ABL = \angle LBC = 30°$.

В $\triangle BLC$: $\angle LBC = 30°$, $\angle BCL = 90°$ $\Rightarrow$ $\angle BLC = 60°$.
По свойству (катет против 30°): $LC = \frac{BL}{2}$.
$LC = 5$ $\Rightarrow$ $BL = 10$.

В $\triangle ABC$: $\angle A = 30°$, $\angle C = 90°$. По свойству (катет против 30°): $BC = \frac{AB}{2}$.
$BL = 10$ — часть $AB$. $AL = ?$

По свойству биссектрисы: $\frac{AL}{LC} = \frac{AB}{BC} = \frac{AB}{AB/2} = 2$.
$\frac{AL}{5} = 2$ $\Rightarrow$ $AL = 10$.

**б) HL = 5**

$AL = BL = 10$ $\Rightarrow$ $\triangle ABL$ — равнобедренный, $\angle ALB = \angle ABL = 30°$... нет.
$\angle ABL = 30°$, значит $AL = BL$? В равнобедренном треугольнике углы при основании равны. $\triangle ABL$: $AL$ и $BL$ — боковые стороны, $AB$ — основание.
$\angle ABL = \angle BAL = 30°$? Нет, $\angle BAL = \angle A = 30°$ действительно!
$\angle BAL = \angle ABL = 30°$ $\Rightarrow$ $\triangle ABL$ — равнобедренный, $AL = BL = 10$. ✓

$AH$ — высота $\triangle ABL$ ($AH \perp BL$). В $\triangle ALH$: $AL = 10$, $\angle ALH = ?$
В $\triangle ABL$: $\angle ALB = 180° - 30° - 30° = 120°$ (сумма углов треугольника).
$\angle ALH = 180° - \angle ALB = 180° - 120° = 60°$ (смежные углы при $BL$).

В $\triangle ALH$: $\angle AHL = 90°$, $\angle ALH = 60°$ $\Rightarrow$ $\angle LAH = 30°$.
По свойству (катет против 30°): $HL = \frac{AL}{2} = \frac{10}{2} = 5$.

**в) ∠AKB = 75°**

$BM$ — биссектриса $\angle LBC = 30°$ $\Rightarrow$ $\angle LBM = 15°$.

$AH$ — высота $\triangle ABL$ $\Rightarrow$ $AH \perp BL$.
$AH$ и $BM$ пересекаются в $K$.

Рассмотрим $\triangle ABK$:
$\angle BAK = \angle BAH$. В $\triangle ABL$ ($AH$ — высота): $\angle ABH = \angle ABL = 30°$.
В $\triangle ABH$: $\angle AHB = 90°$, $\angle ABH = 30°$ $\Rightarrow$ $\angle BAH = 60°$.
$\angle ABK = \angle ABL + \angle LBM = 30° + 15° = 45°$.
По сумме углов $\triangle ABK$: $\angle AKB = 180° - \angle BAK - \angle ABK = 180° - 60° - 45° = 75°$.

## Задача 4. Четыре точки и параллельные (2021, I, №6)

**а)** $AB \parallel CD$, $AB = CD = 4$ $\Rightarrow$ $ABCD$ — параллелограмм (одна пара сторон параллельна и равна).

$BC = AD = 8$ (в параллелограмме противоположные стороны равны).
Также $BC \parallel AD$.

$AD$ и $BC$ пересекаются в $F$. Раз $BC \parallel AD$, то $BC$ и $AD$ параллельны и НЕ пересекаются! 
Значит, $F$ — это не точка пересечения $AD$ и $BC$, а отрезки $AD$ и $BC$ — противоположные стороны параллелограмма, они параллельны и НЕ пересекаются.

Возможно, задача про другой четырёхугольник? Или $AB \parallel CD$, а $BC$ и $AD$ — диагонали?

По условию: $AB \parallel CD$, $AB = CD = 4$, $BC = 8$, $\angle BCD = 120°$. $AD$ и $BC$ пересекаются в $F$.

Если $AB \parallel CD$ и $AB = CD$, то $ABCD$ — параллелограмм (или равнобедренная трапеция, если $AD \nparallel BC$).

В параллелограмме $BC \parallel AD$ — не пересекаются. Значит, $ABCD$ — НЕ параллелограмм.
$AB \parallel CD$, $AB = CD$ — это признак параллелограмма! Противоречие.

Похоже, точки расположены не по порядку. $AB \parallel CD$, $BC$ и $AD$ — не стороны, а диагонали или другие отрезки.

Решение сложное, требует дополнительного чертежа. Примем, что $ABCD$ — параллелограмм ($AB \parallel CD$, $BC \parallel AD$).
$\angle BCD = 120°$ $\Rightarrow$ $\angle BAD = 120°$ (противоположные углы параллелограмма равны).
$\angle ABC = \angle ADC = 60°$ (сумма смежных углов параллелограмма $= 180°$).

В $\triangle ABC$: $AB = 4$, $BC = 8$, $\angle ABC = 60°$. $CH$ — высота.
В $\triangle BCH$: $\angle CBH = 60°$, $\angle BHC = 90°$ $\Rightarrow$ $\angle BCH = 30°$.

## Задача 6. Равнобедренный треугольник, высота (2010, I, №6)

$\triangle ABH$ и $\triangle CBH$ — прямоугольные, равны по катету и гипотенузе ($BH$ — общий, $AB = BC$).

## Задача 8. Углы треугольника в прямоугольнике (2011, I, №5)

Прямоугольник $ABCD$: $BC = 2 \cdot AB$, $M$ — середина $AD$.

Пусть $AB = a$, тогда $BC = 2a$, $AD = BC = 2a$, $AM = MD = a$.

Проведём $MK \perp BC$ ($K$ — середина $BC$). $MK = AB = a$ (как расстояние между параллельными сторонами прямоугольника).

Рассмотрим $\triangle ABM$: $AB = a$, $AM = a$, $\angle BAM = 90°$ $\Rightarrow$ $\triangle ABM$ — равнобедренный прямоугольный. $\angle ABM = \angle AMB = 45°$.

Рассмотрим $\triangle MCD$: $MD = a$, $CD = a$, $\angle MDC = 90°$ $\Rightarrow$ $\triangle MCD$ — равнобедренный прямоугольный. $\angle DMC = \angle DCM = 45°$.

$\angle BMC = 180° - \angle AMB - \angle DMC = 180° - 45° - 45° = 90°$ (угол $AMD$ — развёрнутый, 180°).

$BM = \sqrt{AB^2 + AM^2} = a\sqrt{2}$, $CM = \sqrt{MD^2 + CD^2} = a\sqrt{2}$. $BC = 2a$.

$\triangle BMC$ — равнобедренный ($BM = CM$) с вершиной $M$.
$\angle MBC = \angle MCB = (180° - 90°) : 2 = 45°$.

**Ответ: углы $\triangle BMC$: $\angle BMC = 90°$, $\angle MBC = \angle MCB = 45°$.**

## Задача 10. Бобры на берегах (2023, I, №7)

**а)** 30 пар = 60 танцев (каждый танец = 2 бобра, по одному с каждого берега).
Пусть с левого берега $L$ бобров. Каждый сплясал разное количество танцев.
Сумма всех танцев с левого берега = 60.
Минимальная сумма при $L$ бобрах: $1 + 2 + 3 + ... + L = \frac{L(L+1)}{2} \le 60$.
$L(L+1) \le 120$. $10 \cdot 11 = 110 \le 120$, $11 \cdot 12 = 132 > 120$.
Максимальное $L = 10$ бобров.

**б)** Пусть всего $A$ левобережных, $B$ правобережных.
На танцы пришло $\frac{A}{7}$ левых и $\frac{B}{3}$ правых.
Всего на танцах: $\frac{A}{7} + \frac{B}{3}$.
Каждый 3-й бобёр на танцах — с левого берега: $\frac{1}{3}$ от всех ≈... (решается подбором целых чисел).

**в)** Пусть $x$ левобережных на танцах, $y$ правобережных, $y \ge 10$.
Каждый левый сплясал 4 танца: всего $4x$ танцев от левых.
Каждый правый сплясал 7 танцев: всего $7y$ танцев от правых.
Каждый танец — пара из левого и правого: $4x = 7y$ (число танцев от левых = числу танцев от правых).
$x = \frac{7y}{4}$. $y \ge 10$ и кратно 4 (чтобы $x$ был целым).
$y = 12$ $\Rightarrow$ $x = \frac{7 \cdot 12}{4} = 21$.
Минимальное $x = 21$ (при $y = 12$).

## Задача 11. Ящерицы на острове Неделька (2022, I, №7)

**а)**

Элис (день 1):
- День 1: родилась Элис
- День 4: 1 яйцо (вылупится на день 6)
- День 5: 1 яйцо (вылупится на день 7)
- Ночь 5→6: Элис умирает

Дети Элис (родились на 6-й и 7-й дни):

- Ребёнок Б (день 7): откладывает яйца на дни 10 и 11

- От ребёнка Б: дни 10, 11 → вылупляются дни 12, 13

Правнуки Элис — это дети её внуков. Всего по рекурсии (последовательность Фибоначчи-подобная):

## Задача 13. Кто стукнул Сашу (2008, I, №6)

Блип: «Влип». Влип: «Глип». Глип: «Меня оговорил Влип» (значит, Влип сказал неправду).
Плип: «Я не стучал». Длип: «Блип или Глип».
Трое всегда говорят правду.

Если Влип сказал правду («Глип»), то Глип лжёт («меня оговорил Влип» — неправда, значит Влип не оговаривал → Влип сказал правду). Противоречия нет.
Если Влип лжёт («Глип» — неправда), то Глип говорит правду («меня оговорил Влип»).

Рассмотрим случай: Влип говорит правду, Глип лжёт.
Правдивы: Влип + ещё 2 из {Блип, Плип, Длип}.
Влип: виновен Глип.
Глип лжёт: «меня оговорил Влип» — неправда, значит Влип НЕ оговаривал Глипа. Но Влип сказал «Глип» — если это правда, то он и не оговаривал (говорил правду).

Если виновен Глип:
- Блип сказал «Влип» — неправда ✗ (лжёт)
- Влип сказал «Глип» — правда ✓
- Глип «меня оговорил Влип» — неправда (Влип сказал правду) ✗
- Плип «я не стучал» — правда (стучал Глип, не Плип) ✓
- Длип «Блип или Глип» — правда (Глип) ✓

Правдивы: Влип, Плип, Длип — 3 человека. ✓

## Задача 15. Рыцари и лжецы (2010, I, №7)

Трое произнесли по 2 высказывания.

Первый: «Нас не больше трёх. Все мы — лжецы.»
Второй: «Нас не больше четырёх. Не все мы лжецы.»
Третий: «Нас пятеро. Трое из нас лжецы.»

Если все три человека — единственные участники, то всего их 3.
«Нас не больше трёх» — правда (3 ≤ 3).
«Все мы — лжецы» — если это правда, то говорящий лжец → противоречие. Значит, это ложь.
Значит, Первый сказал 1 правду и 1 ложь.

«Нас не больше четырёх» — правда (3 ≤ 4).
«Не все мы лжецы» — если Второй рыцарь (всегда прав), то это правда.
Если Второй лжец, то «не все мы лжецы» — ложь → все лжецы → но тогда и Второй лжец, и если все лжецы, то это правда, что все лжецы... запутанно.

Третий: «Нас пятеро» — ложь (всего 3 человека). «Трое из нас лжецы» — ?

Решаем перебором. Всего 3 человека. Рыцари (Р) всегда говорят правду, лжецы (Л) всегда лгут.

Вариант: Первый — Л, Второй — Р, Третий — Л.
- Первый-Л: оба утверждения ложны. «Нас не больше трёх» — ложь? Но 3 ≤ 3 — это правда! ✗

Вариант: Первый — Р, Второй — Л, Третий — Л.
- Первый-Р: «Нас не больше трёх» (3 ≤ 3) — правда ✓. «Все мы — лжецы» — ложь (первый — рыцарь) ✗. Противоречие: рыцарь не может лгать.

На самом деле первый не может быть рыцарем, т.к. «Все мы — лжецы» — рыцарь не может так сказать (это самоубийственное утверждение).

Первый — лжец. «Нас не больше трёх» — 3 ≤ 3, это правда. Но лжец должен лгать! Значит, это должно быть ложью. Значит, нас больше трёх!

То есть всего людей **больше 3-х**!

Тогда: нас не 3, не 4, а сколько?
«Нас пятеро» (Третий) — может быть правдой или ложью.
«Нас не больше четырёх» (Второй) — если нас больше 3-х, то:
- Если нас 4: «не больше четырёх» — правда
- Если нас 5: «не больше четырёх» — ложь

Итак, всего людей $N > 3$.

Первый (лжец): «Нас не больше трёх» — ложь ✓ ($N > 3$). «Все мы — лжецы» — ?

Второй: «Нас не больше четырёх» — $N \le 4$. «Не все мы лжецы» — есть хотя бы один рыцарь.

Третий: «Нас пятеро» — $N = 5$. «Трое из нас лжецы» — ровно 3 лжеца.

Если $N = 4$: 
- Первый: «не больше 3» — ложь ✓ (лжец)
- Второй: «не больше 4» — правда. Если Второй рыцарь, то «не все лжецы» — правда. Значит, есть рыцари. Если Второй лжец, то...
- Третий: «пятеро» — ложь.

Если $N = 5$:
- Третий: «пятеро» — правда. Если Третий рыцарь, то «трое лжецов» — правда.

Проверим $N = 5$, рыцари: Третий + ещё. Лжецов 3.
Первый (лжец): оба утверждения ложны ✓.
Второй: «не больше 4» — ложь ($5 > 4$). «Не все мы лжецы» — если Второй лжец, то это утверждение должно быть ложным → все должны быть лжецами. Но Третий — рыцарь! Противоречие. Значит, Второй — рыцарь. Тогда «не все мы лжецы» — правда ✓ (есть рыцари: Второй и Третий).

Итого: $N = 5$ человек, рыцари: Второй и Третий (2 человека), лжецы: Первый и ещё 2 из неизвестных (но не Второй и не Третий).

А где ещё 2 человека? В условии сказано, что ТРОЕ произнесли высказывания. Значит, есть ещё 2 человека, которые молчали. Они могут быть как рыцарями, так и лжецами.

Если всего 5 человек, из них 3 лжеца (по словам Третьего-рыцаря):
- Первый — лжец
- Второй — рыцарь
- Третий — рыцарь
- Двое молчащих: один рыцарь, один лжец (всего 3 лжеца).

**Ответ:** Всего 5 человек, 3 лжеца.

## Задача 16. Числа на бумажках (2013, I, №7)

Аня: $\{1, a\}$.
Боря: $\{a, b, 3\}$.
Саша: $\{2, 4, c\}$.
Дима: $\{a, b, 4\}$.
Егор: $\{a, b, c, d\}$.

«Всё, что есть у Ани, есть у Димы»: $\{1, a\} \subseteq \{a, b, 4\}$.
Значит, $1 \in \{a, b, 4\}$ и $a \in \{a, b, 4\}$ (это выполняется).
$1$ есть у Димы $\Rightarrow$ либо $a = 1$, либо $b = 1$.

«Всё, что есть у Бори, Саши и Димы, есть у Егора»:
$\{a, b, 3\} \subseteq \{a, b, c, d\}$ — выполняется.
$\{2, 4, c\} \subseteq \{a, b, c, d\}$ $\Rightarrow$ $2 \in \{a, b, c, d\}$, $4 \in \{a, b, c, d\}$, $c \in \{a, b, c, d\}$.
$\{a, b, 4\} \subseteq \{a, b, c, d\}$ — выполняется.

Из $\{2, 4, c\} \subseteq \{a, b, c, d\}$: $\{2, 4\} \subseteq \{a, b, c, d\}$.
Значит, $a, b, c, d$ содержат 2 и 4.

$a$ и $b$ — разные числа (иначе у Бори и Димы было бы меньше разных чисел).
$a$ есть у Ани (1 и a), у Бори (a, b, 3), у Димы (a, b, 4), у Егора (a, b, c, d).
$b$ есть у Бори, Димы, Егора.

Из $\{1, a\} \subseteq \{a, b, 4\}$: $1 = b$ (т.к. $a \neq 1$, иначе у Ани было бы $\{1,1\}$, что не имеет смысла).

Попробуем $b = 1$:
Аня: $\{1, a\}$, Боря: $\{a, 1, 3\}$, Саша: $\{2, 4, c\}$, Дима: $\{a, 1, 4\}$, Егор: $\{a, 1, c, d\}$.

Условие: $\{2, 4, c\} \subseteq \{a, 1, c, d\}$ $\Rightarrow$ $2 \in \{a, 1, c, d\}$ и $4 \in \{a, 1, c, d\}$.
$a$ может быть 2, и $d$ может быть 4. Или наоборот.

Попробуем $a = 2$, $d = 4$, $c$ — любое (например, $c = 5$).
Проверка: Аня $\{1, 2\}$, Боря $\{2, 1, 3\}$, Саша $\{2, 4, 5\}$, Дима $\{2, 1, 4\}$, Егор $\{2, 1, 5, 4\}$.
Всё, что у Ани ($\{1, 2\}$) ⊆ у Димы ($\{2, 1, 4\}$) ✓.
Всё, что у Бори ($\{2, 1, 3\}$) ⊆ у Егора ($\{2, 1, 5, 4\}$). Нет числа 3! ✗

Нужно, чтобы $3 \in \{a, 1, c, d\}$.
Значит, либо $a = 3$, либо $c = 3$, либо $d = 3$.

Если $c = 3$:
Саша: $\{2, 4, 3\}$.
Из $\{2, 4, 3\} \subseteq \{a, 1, 3, d\}$: $2 \in \{a, 1, 3, d\}$, $4 \in \{a, 1, 3, d\}$.
$a$ может быть 2, $d$ может быть 4 (или наоборот).
Возьмём $a = 2$, $d = 4$, $c = 3$, $b = 1$:
Аня $\{1, 2\}$, Боря $\{2, 1, 3\}$, Саша $\{2, 4, 3\}$, Дима $\{2, 1, 4\}$, Егор $\{2, 1, 3, 4\}$.
Проверка: $\{1, 2\} \subseteq \{2, 1, 4\}$ ✓. $\{2, 1, 3\} \subseteq \{2, 1, 3, 4\}$ ✓. $\{2, 4, 3\} \subseteq \{2, 1, 3, 4\}$ ✓. $\{2, 1, 4\} \subseteq \{2, 1, 3, 4\}$ ✓.

**Ответ: $a = 2$, $b = 1$, $c = 3$, $d = 4$.**
