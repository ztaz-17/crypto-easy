## Задача 1. Бег по стадиону (2025, I, №5)

$v_K = 2$ км/ч $= \frac{2000}{60} = \frac{100}{3}$ м/мин. $v_E = 3v_K = 100$ м/мин.

План: встретиться на противоположной стороне — каждый пробегает $L/2$. Плановое время: $t_K^{план} = \frac{L/2}{100/3} = \frac{3L}{200}$, $t_E^{план} = \frac{L/2}{100} = \frac{L}{200}$. Разница $= \frac{L}{100} = 3$ мин $\Rightarrow$ $L_{план} = 300$ м.

Факт: Эрвин пробежал $\frac{L}{2} + 50$ м за $t_E = \frac{L/2 + 50}{100}$. Катерина бежала $t_K = t_E + 3$ минут. Сумма расстояний $= L$:
$\frac{100}{3}(t_E + 3) + (\frac{L}{2} + 50) = L$.
$\frac{100}{3}(\frac{L/2 + 50}{100} + 3) + \frac{L}{2} + 50 = L$.
$\frac{L/2 + 50}{3} + 100 + \frac{L}{2} + 50 = L$.
$\frac{L/2 + 50}{3} + \frac{L}{2} + 150 = L$.
Умножаем на $3$: $(L/2 + 50) + \frac{3L}{2} + 450 = 3L$.
$2L + 500 = 3L$ $\Rightarrow$ $L = 500$ м.

**Ответ: $500$ м.**

## Задача 2. Эрвин ведёт Катерину (2023, I, №5)

$v_1 = 2{,}5$ км/ч $= 2500$ м/ч. После остановки: $v_2 = 2500 + 500 = 3000$ м/ч $= 3$ км/ч.

Задержка $= 4$ мин $= \frac{1}{15}$ ч. Пусть $S$ (км) — расстояние.

План: $T_{план} = \frac{S}{2{,}5} = \frac{2S}{5}$ ч.

Факт: $t_1 = \frac{S/9}{2{,}5} = \frac{2S}{45}$, $t_{задерж} = \frac{1}{15}$, $t_2 = \frac{8S/9}{3} = \frac{8S}{27}$.
$T_{факт} = \frac{2S}{45} + \frac{1}{15} + \frac{8S}{27}$.
$T_{факт} = T_{план}$: $\frac{2S}{45} + \frac{1}{15} + \frac{8S}{27} = \frac{2S}{5}$.
Общий знаменатель $135$: $\frac{6S + 9 + 40S}{135} = \frac{54S}{135}$ $\Rightarrow$ $46S + 9 = 54S$ $\Rightarrow$ $8S = 9$ $\Rightarrow$ $S = \frac{9}{8} = 1{,}125$ км.

Средняя скорость: $v_{ср} = \frac{S}{T_{план}} = \frac{1{,}125}{0{,}45} = 2{,}5$ км/ч.

**Ответ: $1{,}125$ км ($1125$ м), $v_{ср} = 2{,}5$ км/ч.**

## Задача 3. Эрвин из школы (2021, I, №5)

$v_1 = 4$ км/ч, $v_2 = 150$ м/мин $= \frac{150\cdot60}{1000} = 9$ км/ч. $\Delta t = 15$ мин $= 0{,}25$ ч.

Пусть $S$ (км) — путь. План: $T_{план} = \frac{S}{4}$.

Факт: $t_1 = \frac{S/3}{4} = \frac{S}{12}$, $t_2 = \frac{2S/3}{9} = \frac{2S}{27}$.

$T_{план} - T_{факт} = 0{,}25$: $\frac{S}{4} - (\frac{S}{12} + \frac{2S}{27}) = \frac14$.
$\frac{S}{4} - \frac{S}{12} - \frac{2S}{27} = \frac14$. Общий знаменатель $108$: $\frac{27S - 9S - 8S}{108} = \frac14$ $\Rightarrow$ $\frac{10S}{108} = \frac14$ $\Rightarrow$ $10S = 27$ $\Rightarrow$ $S = 2{,}7$ км.

**Ответ: $2{,}7$ км.**

## Задача 4. Эрвин в садике (2014, I, №5)

Расстояние $3$ км. Эрвин: $v_1 = 5$ км/ч (туда), $v_2 = 6$ км/ч (обратно). Катерина: $v_3 = 4$ км/ч (домой).

Пусть $x$ (км) прошёл Эрвин до встречи с Катериной. Время Эрвина: $t = \frac{x}{5}$ ч.

Катерина за это время прошла $3 - x$ км (от садика до места встречи). $t = \frac{3-x}{4}$.
$\frac{x}{5} = \frac{3-x}{4}$ $\Rightarrow$ $4x = 15 - 5x$ $\Rightarrow$ $9x = 15$ $\Rightarrow$ $x = \frac{5}{3}$ км.
$t = \frac{5/3}{5} = \frac13$ ч $= 20$ мин.

**Ответ: $20$ мин.**

## Задача 5. Вороны на деревьях (2006, II тур, №1)

Всего $35$ ворон. Пусть $x$ — на берёзе, $y$ — на ольхе. $x + y = 35$.
После того как с берёзы улетело $5$: $x - 5 = 2y$.
$x = 2y + 5$. Подставляем: $2y + 5 + y = 35$ $\Rightarrow$ $3y = 30$ $\Rightarrow$ $y = 10$, $x = 25$.

**Ответ: на берёзе $25$, на ольхе $10$.**

## Задача 6. Рекс и число (2007, I, №3)

Пусть $x$ — задуманное число. По заданию: $4x + 15$. По факту: $15x + 4$. Результаты равны:
$4x + 15 = 15x + 4$ $\Rightarrow$ $15 - 4 = 15x - 4x$ $\Rightarrow$ $11 = 11x$ $\Rightarrow$ $x = 1$.

**Ответ: $1$.**

## Задача 7. Рекс вылупился (2008, I, №3)

Рексу $13$ дней, Саше $3$ дня. Через $t$ дней Рекс станет втрое старше Саши:
$13 + t = 3(3 + t)$ $\Rightarrow$ $13 + t = 9 + 3t$ $\Rightarrow$ $13 - 9 = 3t - t$ $\Rightarrow$ $4 = 2t$ $\Rightarrow$ $t = 2$.

**Ответ: через $2$ дня.**

## Задача 8. Сон Эрвина (2013, I, №5)

Сутки $= 24$ ч. Пусть $d$ — сон днём (когда спит), $n$ — сон ночью. Если не спал днём, то ночью спит $n' = \frac{n}{0{,}8} = \frac{5n}{4}$ (так как $n$ в $1{,}2 = 6/5$ раза меньше $n'$).

Сумма сна $= d + n$. Сумма бодрствования $= 24 - (d + n)$. По условию $d + n = 24 - (d + n)$ $\Rightarrow$ $2(d + n) = 24$ $\Rightarrow$ $d + n = 12$ ч.

Также $n = \frac{n'}{1{,}2} = \frac{5}{6}n'$. Если не спал днём, то ночью спит $n' = 12$ (так как $d=0$, а сумма $12$). Но при $d > 0$: $n' = \frac{6}{5}n$.

Если бы не спал: $0 + n' = 12$ $\Rightarrow$ $n' = 12$. Тогда $n = \frac{5}{6}\cdot12 = 10$. $d = 12 - 10 = 2$.

**Ответ: $2$ часа.**

## Задача 9. Lego детальки (2024, I, №5)

Первый набор: $N_1 = 1000$, прямоугольных $= 0{,}38 \cdot 1000 = 380$, квадратных $= 620$.
Второй набор: квадратных $640$ — это $80\%$ (так как $20\%$ прямоугольных). $N_2 = \frac{640}{0{,}8} = 800$, прямоугольных $= 0{,}2 \cdot 800 = 160$.
Общий: $N = 1800$, прямоугольных $= 380 + 160 = 540$. Процент $= \frac{540}{1800}\cdot100\% = 30\%$.

**Ответ: $30\%$.**

## Задача 10. Умпа и Лумпа (2021, I, №4)

$x$ — грибов у Умпы, $y$ — у Лумпы. $x + y = 130$.
Утром: $0{,}75x$ (Умпа), $0{,}8y$ (Лумпа). $0{,}75x = 1{,}5 \cdot 0{,}8y = 1{,}2y$.
$x = \frac{1{,}2}{0{,}75}y = 1{,}6y$. $1{,}6y + y = 130$ $\Rightarrow$ $2{,}6y = 130$ $\Rightarrow$ $y = 50$, $x = 80$.

**Ответ: Умпа — $80$, Лумпа — $50$.**

## Задача 11. Эрвин строит башню (2012, I, №6)

Эрвин: $p_E = \frac{1}{30}$ башни/мин. Саня в $3$ раза быстрее: $p_S = \frac{3}{30} = \frac{1}{10}$.
Егор в $1{,}5$ раза дольше Сани: $t_{Егор} = 1{,}5 \cdot 10 = 15$ мин, $p_{Егор} = \frac{1}{15}$.
Суммарная скорость: $\frac{1}{30} + \frac{1}{10} + \frac{1}{15} = \frac{1+3+2}{30} = \frac{6}{30} = \frac15$.
Время: $1 : \frac15 = 5$ мин.

**Ответ: $5$ мин.**

## Задача 12. Эрвин съедает тарелку (2012, II, №6)

Эрвин: $p_E = \frac{1}{9}$ тарелки/мин. Егор в $1{,}5$ раза быстрее: $t_Е = \frac{9}{1{,}5} = 6$ мин, $p_Е = \frac{1}{6}$.
Саня в $3$ раза дольше Егора: $t_С = 3 \cdot 6 = 18$ мин, $p_С = \frac{1}{18}$.
Суммарная скорость: $\frac{1}{9} + \frac{1}{6} + \frac{1}{18} = \frac{2+3+1}{18} = \frac{6}{18} = \frac13$.
Время: $1 : \frac13 = 3$ мин.

**Ответ: $3$ мин.**

## Задача 14. Эрвин набирает вес (2010, I, №4)

Пусть $x$ (г) — вес при рождении.
После $1$-го месяца: $1{,}25x$. После $2$-го: $1{,}25x \cdot 1{,}3 = 1{,}625x$.
После $3$-го: $1{,}625x + 1170 = 6240$.
$1{,}625x = 5070$ $\Rightarrow$ $x = \frac{5070}{1{,}625} = 3120$ г.

Прибавка: $\frac{6240 - 3120}{3120}\cdot100\% = \frac{3120}{3120}\cdot100\% = 100\%$.

**Ответ: а) $3120$ г; б) $100\%$.**

## Задача 15. Уровень компота (2007, I, №6)

Пусть начальный уровень $= 1$. После съедания половины персиков уровень $= 1 - \frac13 = \frac23$.
При этом съедено $50\%$ персиков, уровень упал на $\frac13$ от начального.

После съедания половины оставшихся (ещё $25\%$ от исходного) уровень понизится на $\frac13 \cdot \frac12 = \frac16$ от начального уровня.

Новый уровень $= 1 - \frac13 - \frac16 = 1 - \frac12 = \frac12$.
Падение от полученного уровня: $\frac{1/6}{2/3} = \frac{1}{6} \cdot \frac{3}{2} = \frac14$.

**Ответ: на $\frac14$ ($25\%$).**

## Задача 16. Башенки из кубиков (2022, I, №5)

Скорость сборки Катерины: $p_K = \frac{12}{5} = 2{,}4$ куб/мин.
Время на башню из $5$ кубиков: $\frac{5}{2{,}4} = \frac{25}{12}$ мин.
За это время Эрвин строит $2 \cdot 9 = 18$ кубиков $\Rightarrow$ $p_E = \frac{18}{25/12} = \frac{216}{25} = 8{,}64$ куб/мин.

Катерина мешает (разбирает): $p_K^{разб} = 2{,}4$ куб/мин.
Чистая скорость: $p_{чист} = 8{,}64 - 2{,}4 = \frac{216}{25} - \frac{12}{5} = \frac{216-60}{25} = \frac{156}{25}$ куб/мин.

Время на $26$ кубиков: $t = \frac{26}{156/25} = 26 \cdot \frac{25}{156} = \frac{650}{156} = \frac{25}{6} = 4\frac16$ мин $= 4$ мин $10$ сек.

**Ответ: $4$ мин $10$ сек.**
