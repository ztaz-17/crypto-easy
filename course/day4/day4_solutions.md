## Задача 1 (2024, I, №1в)

**Система:**
\[
\begin{cases}
x + y = 7,\\
x - y = 18
\end{cases}
\]

### Решение

**Шаг 1.** Складываем уравнения:

\[
(x + y) + (x - y) = 7 + 18 \quad\Rightarrow\quad 2x = 25 \quad\Rightarrow\quad x = \frac{25}{2}.
\]

**Шаг 2.** Вычитаем из первого уравнения второе (или подставляем \(x\)):

\[
(x + y) - (x - y) = 7 - 18 \quad\Rightarrow\quad 2y = -11 \quad\Rightarrow\quad y = -\frac{11}{2}.
\]

**Ответ:** \(\displaystyle \left(\frac{25}{2};\; -\frac{11}{2}\right)\).

## Задача 2 (2024, I, №3а)

\[
(x - 2)(x^2 + 5x + 6) = (2x - 4)(x^2 + 3x + 3)
\]

### Решение

**Шаг 1.** Замечаем, что \(2x - 4 = 2(x - 2)\). Переносим всё влево:

\[
(x - 2)(x^2 + 5x + 6) - 2(x - 2)(x^2 + 3x + 3) = 0.
\]

**Шаг 2.** Выносим общий множитель \((x - 2)\):

\[
(x - 2)\bigl[(x^2 + 5x + 6) - 2(x^2 + 3x + 3)\bigr] = 0.
\]

**Шаг 3.** Упрощаем выражение в скобках:

\[
\begin{aligned}
(x^2 + 5x + 6) - 2(x^2 + 3x + 3) &=
x^2 + 5x + 6 - 2x^2 - 6x - 6 = \\
&= -x^2 - x = -x(x + 1).
\end{aligned}
\]

**Шаг 4.** Получаем:

\[
(x - 2) \cdot (-x) \cdot (x + 1) = 0.
\]

**Шаг 5.** Произведение равно нулю, когда хотя бы один множитель равен нулю:

\[
x - 2 = 0 \quad\Rightarrow\quad x = 2,
\]
\[
-x = 0 \quad\Rightarrow\quad x = 0,
\]
\[
x + 1 = 0 \quad\Rightarrow\quad x = -1.
\]

**Ответ:** \(-1;\; 0;\; 2\).

## Задача 3 (2024, I, №3б) — рациональное уравнение

\[
\frac{x^2 - 2x}{x - 2} + \frac{x^2 + 4x + 4}{x + 2} = \frac{x^2 - 16}{x - 4}
\]

### Решение

**Шаг 1.** ОДЗ (знаменатели не равны нулю):

\[
x - 2 \neq 0,\quad x + 2 \neq 0,\quad x - 4 \neq 0 \quad\Rightarrow\quad
x \neq 2,\; x \neq -2,\; x \neq 4.
\]

**Шаг 2.** Упрощаем каждую дробь:

\[
\frac{x^2 - 2x}{x - 2} = \frac{x(x - 2)}{x - 2} = x \quad (x \neq 2),
\]
\[
\frac{x^2 + 4x + 4}{x + 2} = \frac{(x + 2)^2}{x + 2} = x + 2 \quad (x \neq -2),
\]
\[
\frac{x^2 - 16}{x - 4} = \frac{(x - 4)(x + 4)}{x - 4} = x + 4 \quad (x \neq 4).
\]

**Шаг 3.** Подставляем в уравнение:

\[
x + (x + 2) = x + 4.
\]

**Шаг 4.** Решаем:

\[
2x + 2 = x + 4 \quad\Rightarrow\quad x = 2.
\]

**Шаг 5.** Проверяем ОДЗ: \(x = 2\) не подходит, так как \(x \neq 2\).

**Ответ:** \(\varnothing\) (нет решений; \(x = 4\) также выколот по ОДЗ, но корень \(x = 2\) оказался недопустимым).

## Задача 4 (2025, I, №3а)

\[
(x^2 - x + 1)(x - 3) + x(x - x^2) = (1 - x)(3x + 2)
\]

### Решение

**Шаг 1.** Раскрываем скобки в левой части:

\[
\begin{aligned}
(x^2 - x + 1)(x - 3) &=
x^3 - 3x^2 - x^2 + 3x + x - 3 = \\
&= x^3 - 4x^2 + 4x - 3,
\end{aligned}
\]
\[
x(x - x^2) = x^2 - x^3.
\]

Складываем:

\[
(x^3 - 4x^2 + 4x - 3) + (x^2 - x^3) = -3x^2 + 4x - 3.
\]

**Шаг 2.** Раскрываем скобки в правой части:

\[
(1 - x)(3x + 2) = 3x + 2 - 3x^2 - 2x = -3x^2 + x + 2.
\]

**Шаг 3.** Получаем уравнение:

\[
-3x^2 + 4x - 3 = -3x^2 + x + 2.
\]

**Шаг 4.** Переносим всё в одну сторону:

\[
-3x^2 + 4x - 3 + 3x^2 - x - 2 = 0 \quad\Rightarrow\quad 3x - 5 = 0.
\]

**Шаг 5.** Решаем:

\[
x = \frac{5}{3}.
\]

**Ответ:** \(\displaystyle \frac{5}{3}\).

## Задача 5 (2023, I, №4а)

\[
(11x + 4x^2)(x^2 - 49)(x + 3) = 0
\]

### Решение

**Шаг 1.** Раскладываем на множители каждую скобку:

\[
11x + 4x^2 = x(11 + 4x),
\]
\[
x^2 - 49 = (x - 7)(x + 7).
\]

**Шаг 2.** Полное разложение:

\[
x(11 + 4x)(x - 7)(x + 7)(x + 3) = 0.
\]

**Шаг 3.** Произведение равно нулю, когда хотя бы один множитель равен нулю:

\[
x = 0,
\]
\[
11 + 4x = 0 \quad\Rightarrow\quad x = -\frac{11}{4},
\]
\[
x - 7 = 0 \quad\Rightarrow\quad x = 7,
\]
\[
x + 7 = 0 \quad\Rightarrow\quad x = -7,
\]
\[
x + 3 = 0 \quad\Rightarrow\quad x = -3.
\]

**Ответ:** \(-7;\; -3;\; -\dfrac{11}{4};\; 0;\; 7\).

## Задача 6 (2023, I, №4б)

\[
\frac{x^2 - 2x - 3}{x^2 - 1} + 1 = \frac{5x - 4}{x - 1}
\]

### Решение

**Шаг 1.** ОДЗ (знаменатели не равны нулю):

\[
x^2 - 1 \neq 0 \quad\Rightarrow\quad x \neq \pm 1,
\]
\[
x - 1 \neq 0 \quad\Rightarrow\quad x \neq 1.
\]

Итого: \(x \neq \pm 1\).

**Шаг 2.** Разложим знаменатели:

\[
x^2 - 1 = (x - 1)(x + 1).
\]

Уравнение примет вид:

\[
\frac{x^2 - 2x - 3}{(x - 1)(x + 1)} + 1 = \frac{5x - 4}{x - 1}.
\]

**Шаг 3.** Умножаем обе части на \((x - 1)(x + 1)\) (при \(x \neq \pm 1\)):

\[
x^2 - 2x - 3 + (x - 1)(x + 1) = (5x - 4)(x + 1).
\]

**Шаг 4.** Раскрываем скобки:

\[
\begin{aligned}
x^2 - 2x - 3 + (x^2 - 1) &= (5x - 4)(x + 1),\\
2x^2 - 2x - 4 &= 5x^2 + 5x - 4x - 4,\\
2x^2 - 2x - 4 &= 5x^2 + x - 4.
\end{aligned}
\]

**Шаг 5.** Переносим всё в правую часть:

\[
0 = 5x^2 + x - 4 - 2x^2 + 2x + 4 = 3x^2 + 3x.
\]

**Шаг 6.** Выносим общий множитель:

\[
3x(x + 1) = 0.
\]

**Шаг 7.** Получаем корни:

\[
x = 0 \quad\text{или}\quad x = -1.
\]

**Шаг 8.** Проверяем ОДЗ: \(x = -1\) не подходит (\(x \neq -1\)). \(x = 0\) — допустим.

**Ответ:** \(0\).
