## Задача 1 (2006, I, №2)

**а)** $(2x + 7) \cdot 2 = 14$ $\Rightarrow$ $2x + 7 = 7$ $\Rightarrow$ $2x = 0$ $\Rightarrow$ $x = 0$.

**б)** $(9x - 3) : 3 - (2x - 4) = 0$ $\Rightarrow$ $3x - 1 - 2x + 4 = 0$ $\Rightarrow$ $x + 3 = 0$ $\Rightarrow$ $x = -3$.

**Ответ: а) $0$; б) $-3$.**

## Задача 2 (2007, I, №2)

$9x^2 - 6x + 1 = (1 - 3x)(2x + 9)$.
Слева: $(3x-1)^2$. Справа: $(1-3x)(2x+9) = -(3x-1)(2x+9)$.
$(3x-1)^2 = -(3x-1)(2x+9)$.
Переносим: $(3x-1)^2 + (3x-1)(2x+9) = 0$ $\Rightarrow$ $(3x-1)[(3x-1)+(2x+9)] = 0$ $\Rightarrow$ $(3x-1)(5x+8) = 0$.
$x = \frac13$ или $x = -\frac85 = -1.6$.

**Ответ: $\frac13$; $-\frac85$.**

## Задача 3 (2010, I, №2)

$\frac{(2.4 - 7x)(x - 1.2)}{1.2 - x} = \frac{(2x + 3.2)(2x + 0.6)}{x + 1.6}$.

ОДЗ: $x \ne 1.2$, $x \ne -1.6$.

В левой части: $x - 1.2 = -(1.2 - x)$, сокращаем: $\frac{(2.4 - 7x)(-1)}{1} = 7x - 2.4$.

Правая часть: $(2x+3.2)(2x+0.6) : (x+1.6)$. Заметим $2x+3.2 = 2(x+1.6)$.
$\frac{2(x+1.6)(2x+0.6)}{x+1.6} = 2(2x+0.6) = 4x + 1.2$.

Уравнение: $7x - 2.4 = 4x + 1.2$ $\Rightarrow$ $3x = 3.6$ $\Rightarrow$ $x = 1.2$ — не подходит по ОДЗ.

**Ответ: нет решений.**

## Задача 4 (2010, II, №2)

$\frac{(3x+1.5)(x+0.7)}{x+0.5} = \frac{(2.3-x)(2x+2.9)}{x-2.3}$.

ОДЗ: $x \ne -0.5$, $x \ne 2.3$.

В левой части: $3x+1.5 = 3(x+0.5)$ $\Rightarrow$ $\frac{3(x+0.5)(x+0.7)}{x+0.5} = 3(x+0.7) = 3x + 2.1$.

В правой части: $2x+2.9 = 2(x+1.45)$. $2.3-x = -(x-2.3)$.
$\frac{-(x-2.3) \cdot 2(x+1.45)}{x-2.3} = -2(x+1.45) = -2x - 2.9$.

Уравнение: $3x + 2.1 = -2x - 2.9$ $\Rightarrow$ $5x = -5$ $\Rightarrow$ $x = -1$.

**Ответ: $-1$.**

## Задача 5 (2011, I, №2)

$\frac{(x^2 - 2x + 1)(3 - 2x)}{x - 1} = 0$. ОДЗ: $x \ne 1$.
$x^2 - 2x + 1 = (x-1)^2$.
$\frac{(x-1)^2(3-2x)}{x-1} = (x-1)(3-2x) = 0$.
$x = 1$ (не подходит по ОДЗ) или $x = \frac32$.

**Ответ: $\frac32$.**

## Задача 10 (2013, I, №4)

$(2x+7)(3x-1) - (5x-1)(x+3) = (x+1)^2$.
$(6x^2 - 2x + 21x - 7) - (5x^2 + 15x - x - 3) = x^2 + 2x + 1$.
$(6x^2 + 19x - 7) - (5x^2 + 14x - 3) = x^2 + 2x + 1$.
$6x^2 + 19x - 7 - 5x^2 - 14x + 3 = x^2 + 2x + 1$.
$x^2 + 5x - 4 = x^2 + 2x + 1$ $\Rightarrow$ $3x = 5$ $\Rightarrow$ $x = \frac53$.

**Ответ: $\frac53$.**

## Задача 11 (2024, I, №3а)

$(x-2)(x^2+5x+6) = (2x-4)(x^2+3x+3)$.
$2x-4 = 2(x-2)$. Переносим: $(x-2)(x^2+5x+6) - 2(x-2)(x^2+3x+3) = 0$.
$(x-2)[(x^2+5x+6) - 2(x^2+3x+3)] = 0$.
$(x-2)[x^2+5x+6 - 2x^2 - 6x - 6] = (x-2)(-x^2 - x) = -(x-2)(x)(x+1) = 0$.
$x = 2$, $x = 0$, $x = -1$.

**Ответ: $-1$; $0$; $2$.**

## Задача 12 (2024, I, №3б)

$\frac{x^2-2x}{x-2} + \frac{x^2+4x+4}{x+2} = \frac{x^2-16}{x-4}$. ОДЗ: $x \ne 2, -2, 4$.
$\frac{x(x-2)}{x-2} = x$ (при $x\ne2$). $\frac{(x+2)^2}{x+2} = x+2$ (при $x\ne-2$). $\frac{(x-4)(x+4)}{x-4} = x+4$ (при $x\ne4$).
Уравнение: $x + (x+2) = x+4$ $\Rightarrow$ $2x+2 = x+4$ $\Rightarrow$ $x = 2$ — не подходит по ОДЗ.

## Задача 13 (2025, I, №3а)

$(x^2 - x + 1)(x - 3) + x(x - x^2) = (1 - x)(3x + 2)$.
Левая часть: $x^3 - 3x^2 - x^2 + 3x + x - 3 + x^2 - x^3 = -3x^2 + 4x - 3$.
Правая часть: $3x + 2 - 3x^2 - 2x = -3x^2 + x + 2$.
$-3x^2 + 4x - 3 = -3x^2 + x + 2$ $\Rightarrow$ $3x = 5$ $\Rightarrow$ $x = \frac53$.

**Ответ: $\frac53$.**

## Задача 14 (2025, I, №3б)

$\frac{x^2-4x+4}{x-2} = \frac{x^2+6x+9}{x+3} - \frac{5x+25}{x+5}$.
ОДЗ: $x \ne 2, -3, -5$.
$\frac{(x-2)^2}{x-2} = x-2$. $\frac{(x+3)^2}{x+3} = x+3$. $\frac{5(x+5)}{x+5} = 5$.
Уравнение: $x-2 = (x+3) - 5$ $\Rightarrow$ $x-2 = x-2$ $\Rightarrow$ тождество.
Все $x$ из ОДЗ подходят: $x \in \mathbb{R} \setminus \{-5, -3, 2\}$.

**Ответ: $x$ — любое, кроме $-5, -3, 2$.**

## Задача 16 (2023, I, №4б)

$\frac{x^2-2x-3}{x^2-1} + 1 = \frac{5x-4}{x-1}$.
ОДЗ: $x \ne \pm 1$.
$x^2-1 = (x-1)(x+1)$. $x^2-2x-3 = (x-3)(x+1)$.
$\frac{(x-3)(x+1)}{(x-1)(x+1)} + 1 = \frac{x-3}{x-1} + 1 = \frac{x-3 + x - 1}{x-1} = \frac{2x-4}{x-1}$.
Уравнение: $\frac{2x-4}{x-1} = \frac{5x-4}{x-1}$.
$2x-4 = 5x-4$ $\Rightarrow$ $-3x = 0$ $\Rightarrow$ $x = 0$.
Проверка ОДЗ: $0 \ne \pm 1$ ✓.

**Ответ: $0$.**
