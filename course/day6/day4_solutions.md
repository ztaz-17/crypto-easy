## Задача 2 (2007, I, №2)

$$9x^2 - 6x + 1 = (1 - 3x)(2x + 9)$$

Слева: $(3x-1)^2$. Справа: $(1-3x)(2x+9) = -(3x-1)(2x+9)$.
$$(3x-1)^2 = -(3x-1)(2x+9)$$
$$(3x-1)^2 + (3x-1)(2x+9) = 0$$
$$(3x-1)[(3x-1)+(2x+9)] = 0$$
$$(3x-1)(5x+8) = 0$$

$x = \frac13$ или $x = -\frac85 = -1{,}6$.

**Ответ:** $\frac13$; $-\frac85$.

## Задача 3 (2010, I, №2)

$$\frac{(2{,}4 - 7x)(x - 1{,}2)}{1{,}2 - x} = \frac{(2x + 3{,}2)(2x + 0{,}6)}{x + 1{,}6}$$

ОДЗ: $x \ne 1{,}2$, $x \ne -1{,}6$.

В левой части: $x - 1{,}2 = -(1{,}2 - x)$, сокращаем:
$$\frac{(2{,}4 - 7x)(-1)}{1} = 7x - 2{,}4$$

В правой части: $2x + 3{,}2 = 2(x + 1{,}6)$:
$$\frac{2(x+1{,}6)(2x+0{,}6)}{x+1{,}6} = 2(2x+0{,}6) = 4x + 1{,}2$$

Уравнение: $7x - 2{,}4 = 4x + 1{,}2$
$$3x = 3{,}6$$
$$x = 1{,}2$$ — не подходит по ОДЗ.

**Ответ:** нет решений.

## Задача 4 (2010, II, №2)

$$\frac{(3x+1{,}5)(x+0{,}7)}{x+0{,}5} = \frac{(2{,}3-x)(2x+2{,}9)}{x-2{,}3}$$

ОДЗ: $x \ne -0{,}5$, $x \ne 2{,}3$.

В левой части: $3x+1{,}5 = 3(x+0{,}5)$:
$$\frac{3(x+0{,}5)(x+0{,}7)}{x+0{,}5} = 3(x+0{,}7) = 3x + 2{,}1$$

В правой части: $2{,}3-x = -(x-2{,}3)$, $2x+2{,}9 = 2(x+1{,}45)$:
$$\frac{-(x-2{,}3) \cdot 2(x+1{,}45)}{x-2{,}3} = -2(x+1{,}45) = -2x - 2{,}9$$

Уравнение: $3x + 2{,}1 = -2x - 2{,}9$
$$5x = -5$$
$$x = -1$$

**Ответ:** $-1$.

## Задача 5 (2011, I, №2)

$$\frac{(x^2 - 2x + 1)(3 - 2x)}{x - 1} = 0$$

ОДЗ: $x \ne 1$.

$x^2 - 2x + 1 = (x-1)^2$.
$$\frac{(x-1)^2(3-2x)}{x-1} = (x-1)(3-2x) = 0$$

$x = 1$ (не подходит по ОДЗ) или $x = \frac32$.

**Ответ:** $\frac32$.

## Задача 8 (2012, II, №1)

$$\frac{1{,}2x - 12}{7} = \frac23 x + 0{,}6$$

$1{,}2 = \frac65$, $0{,}6 = \frac35$.
$$\frac{\frac65 x - 12}{7} = \frac23 x + \frac35$$

Умножаем на $7$:
$$\frac65 x - 12 = 7\left(\frac23 x + \frac35\right) = \frac{14}{3}x + \frac{21}{5}$$

$$\frac65 x - \frac{14}{3}x = \frac{21}{5} + 12$$
$$\frac{18-70}{15}x = \frac{21}{5} + \frac{60}{5}$$
$$-\frac{52}{15}x = \frac{81}{5}$$
$$x = -\frac{81}{5} \cdot \frac{15}{52} = -\frac{81 \cdot 3}{52} = -\frac{243}{52}$$

**Ответ:** $-\frac{243}{52}$.

## Задача 15 (2023, I, №4а)

$$(11x + 4x^2)(x^2 - 49)(x + 3) = 0$$

Выносим $x$:
$$x(11 + 4x)(x-7)(x+7)(x+3) = 0$$

Произведение равно нулю, если любой множитель равен нулю:
$$x = 0,\quad 11 + 4x = 0 \Rightarrow x = -\frac{11}{4},\quad x-7 = 0 \Rightarrow x = 7,$$
$$x+7 = 0 \Rightarrow x = -7,\quad x+3 = 0 \Rightarrow x = -3$$

**Ответ:** $-7$; $-3$; $-\frac{11}{4}$; $0$; $7$.

## Задача 16 (2023, I, №4б)

$$\frac{x^2-2x-3}{x^2-1} + 1 = \frac{5x-4}{x-1}$$

ОДЗ: $x \ne \pm 1$.

$x^2-1 = (x-1)(x+1)$. $x^2-2x-3 = (x-3)(x+1)$.
$$\frac{(x-3)(x+1)}{(x-1)(x+1)} + 1 = \frac{x-3}{x-1} + 1 = \frac{x-3 + x - 1}{x-1} = \frac{2x-4}{x-1}$$

Уравнение:
$$\frac{2x-4}{x-1} = \frac{5x-4}{x-1}$$

При $x \ne 1$:
$$2x-4 = 5x-4$$
$$-3x = 0$$
$$x = 0$$

Проверка ОДЗ: $0 \ne \pm 1$ ✓.

**Ответ:** $0$.
