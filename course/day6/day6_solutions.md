## Задача 1 (2024, I, №1)

**а)** $892^2 - 912^2 = (892-912)(892+912) = (-20)(1804) = -36080$.

**б)** $\frac{4a^2 - b^2}{4a^2 + 4ab + b^2} = \frac{(2a-b)(2a+b)}{(2a+b)^2} = \frac{2a-b}{2a+b}$.

**в)** $\begin{cases} x+y=7 \\ x-y=18 \end{cases}$. Складываем: $2x=25$, $x=12.5$. Вычитаем: $2y=-11$, $y=-5.5$.

**г)** $13\% = 0.13$. $x = 5.2 : 0.13 = 40$.

**е)** Неравенство треугольника: третья сторона $x$ должна быть $|7-4| < x < 7+4$, т.е. $3 < x < 11$. Целые значения: $4$, $5$, $6$, $7$, $8$, $9$, $10$ см.

**Ответ:** а) $-36080$; б) $\frac{2a-b}{2a+b}$; в) $(12.5; -5.5)$; г) $40$; д) $48^\circ, 96^\circ, 36^\circ$; е) $4$, $5$, $6$, $7$, $8$, $9$, $10$ см.

## Задача 2 (2025, I, №1)

**а)** $898^2 = (900-2)^2 = 810000 - 3600 + 4 = 806404$.

**б)** $\frac{p^2}{(p-q)(p+q)} - \frac{2pq - q^2}{p^2-q^2} = \frac{p^2 - (2pq - q^2)}{(p-q)(p+q)} = \frac{p^2 - 2pq + q^2}{(p-q)(p+q)} = \frac{(p-q)^2}{(p-q)(p+q)} = \frac{p-q}{p+q}$.

**в)** $\begin{cases} 2x+3y=7 \\ 3x+2y=3 \end{cases}$. Умножаем 1-е на $3$, 2-е на $2$: $\begin{cases} 6x+9y=21 \\ 6x+4y=6 \end{cases}$. Вычитаем: $5y=15$, $y=3$. $2x+9=7$ $\Rightarrow$ $2x=-2$ $\Rightarrow$ $x=-1$.

**г)** $124\% = 1.24$. $x = 93 : 1.24 = 75$.

**д)** Внешний угол $=120^\circ$ $\Rightarrow$ смежный внутренний $=60^\circ$. Сумма углов $=180^\circ$, один несмежный $=40^\circ$ $\Rightarrow$ третий $=180-60-40=80^\circ$. Углы: $60^\circ$, $40^\circ$, $80^\circ$.

**е)** $AC$ может быть $|5-3| \le AC \le 5+3$, т.е. $2 \le AC \le 8$. На одной прямой возможны два случая: $C$ между $A$ и $B$ ($AC=2$) или $B$ между $A$ и $C$ ($AC=8$).

**Ответ:** а) $806404$; б) $\frac{p-q}{p+q}$; в) $(-1; 3)$; г) $75$; д) $60^\circ, 40^\circ, 80^\circ$; е) $2$ см или $8$ см.

## Задача 12 (2009, I, №6)

Периметр $= 35$. Одна сторона в $3$ раза больше другой.

**Вариант 1:** основание $= x$, боковая $= 3x$ (в равнобедренном треугольнике боковые стороны равны).
$$x + 3x + 3x = 35 \Rightarrow 7x = 35 \Rightarrow x = 5$$
Стороны: $5$, $15$, $15$. Проверка неравенства: $5+15>15$ ✓.

**Вариант 2:** боковая $= x$, основание $= 3x$.
$$3x + x + x = 35 \Rightarrow 5x = 35 \Rightarrow x = 7$$
Стороны: $7$, $7$, $21$. Проверка неравенства: $7+7=14<21$ — не существует!

**Ответ:** $5$, $15$, $15$.

## Задача 13. Внешний угол и биссектриса (адаптировано)

В $\triangle ABC$ внешний угол при вершине $B$ равен $140^\circ$.
Смежный с ним внутренний $\angle B = 180^\circ - 140^\circ = 40^\circ$.

$\angle C = 60^\circ$ (по условию).

Сумма углов треугольника $= 180^\circ$:
$$\angle A = 180^\circ - 40^\circ - 60^\circ = 80^\circ$$

$BL$ — биссектриса $\angle B$, значит $\angle LBC = \frac{\angle B}{2} = \frac{40^\circ}{2} = 20^\circ$.

**Ответ:** $\angle A = 80^\circ$, $\angle B = 40^\circ$, $\angle C = 60^\circ$; $\angle LBC = 20^\circ$.

## Задача 16. Проценты (адаптировано)

Исходный вес: $80$ кг.
После еды ($+15\%$): $80 \cdot 1{,}15 = 92$ кг.
После прогулки ($-15\%$ от нового веса): $92 \cdot 0{,}85 = 78{,}2$ кг.

Вес не вернулся к исходному, потому что $15\%$ от $92$ кг ($= 13{,}8$ кг) больше, чем $15\%$ от $80$ кг ($= 12$ кг). Уменьшение было на бо́льшую величину, чем увеличение, хотя процент был одинаковый.

**Ответ:** $78{,}2$ кг. Из-за разной базы для процентов.
