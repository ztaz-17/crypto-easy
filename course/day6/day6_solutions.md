## 1. Проценты

### Задача 1а (2024, I, №1г)
**Условие:** Найдите число, если 13% от него равны 5.2.

**Решение:**
13% = 0.13. Чтобы найти число по его проценту, делим известную часть на соответствующую десятичную дробь:

x = 5.2 / 0.13 = 520 / 13 = 40.

**Ответ:** 40.

---

### Задача 1б (2025, I, №1г)
**Условие:** Число увеличили на 24% и получили 93. Найдите исходное число.

**Решение:**
Увеличение на 24% означает, что новое число составляет 124% от исходного.
124% = 1.24. Следовательно:

x = 93 / 1.24 = 9300 / 124 = 75.

**Проверка:** 75 + 24% от 75 = 75 + 18 = 93. Верно.

**Ответ:** 75.

## 2. Углы

### Задача 2а (2024, I, №1д)
**Условие:** Углы треугольника относятся как 4:8:3. Найдите углы треугольника.

**Решение:**
Сумма углов треугольника равна 180°. Пусть одна часть равна k. Тогда:
4k + 8k + 3k = 180°
15k = 180°
k = 12°.

Углы треугольника:
- Первый угол: 4 · 12° = 48°.
- Второй угол: 8 · 12° = 96°.
- Третий угол: 3 · 12° = 36°.

**Проверка:** 48° + 96° + 36° = 180°. Верно.

**Ответ:** 48°, 96°, 36°.

---

### Задача 2б (2022, I, №1г)
**Условие:** Найдите смежные углы, если один из них на 30° больше другого.

**Решение:**
Сумма смежных углов равна 180°. Пусть меньший угол равен x. Тогда больший угол равен x + 30°.

x + (x + 30°) = 180°
2x + 30° = 180°
2x = 150°
x = 75° — меньший угол.
x + 30° = 105° — больший угол.

**Ответ:** 75° и 105°.

## 3. Упрощение дробей

### Задача 3а (2024, I, №1б)
**Условие:** Сократите дробь: (4a² − b²) / (4a² + 4ab + b²).

**Решение:**
Разложим числитель и знаменатель на множители:
- Числитель: 4a² − b² = (2a − b)(2a + b) — разность квадратов.
- Знаменатель: 4a² + 4ab + b² = (2a)² + 2·(2a)·b + b² = (2a + b)² — квадрат суммы.

Подставляем:
(4a² − b²) / (4a² + 4ab + b²) = ((2a − b)(2a + b)) / (2a + b)² = (2a − b) / (2a + b).

**Ответ:** (2a − b) / (2a + b).

---

### Задача 3б (дополнительная)
**Условие:** Упростите: p²/(p² − q²) − (2pq − q²)/(p² − q²).

**Решение:**
Приводим к общему знаменателю p² − q²:
(p² − (2pq − q²)) / (p² − q²) = (p² − 2pq + q²) / (p² − q²) = (p − q)² / ((p − q)(p + q)) = (p − q) / (p + q).

**Ответ:** (p − q) / (p + q).

## 4. Разложение на множители

### Задача 4а (2022, I, №1а)
**Условие:** Разложите на множители: 4x² − 4x + 1.

**Решение:**
Замечаем, что это квадрат разности: (a − b)² = a² − 2ab + b².
4x² − 4x + 1 = (2x)² − 2 · (2x) · 1 + 1² = (2x − 1)².

**Ответ:** (2x − 1)².

---

### Задача 4б (2022, I, №1б)
**Условие:** Упростите выражение: 5(a − 3) − 2(a + 2).

**Решение:**
Раскрываем скобки и приводим подобные слагаемые:
5(a − 3) − 2(a + 2) = 5a − 15 − 2a − 4 = 3a − 19.

**Ответ:** 3a − 19.

## 5. Неравенство треугольника и геометрия

### Задача 5а (2024, I, №1е)
**Условие:** В тупоугольном треугольнике две стороны равны 4 см и 7 см. Найдите возможные целые значения третьей стороны.

**Решение:**
Пусть третья сторона равна x см.

1. **Неравенство треугольника (сверху):** Каждая сторона меньше суммы двух других.
   x < 4 + 7 → x < 11.

2. **Неравенство треугольника (снизу):** Каждая сторона больше разности двух других.
   x > 7 − 4 → x > 3.

3. **Условие тупоугольности:** Квадрат стороны, лежащей напротив тупого угла, больше суммы квадратов двух других сторон.
   Предполагая, что тупой угол образован сторонами 4 и 7 (т.е. напротив третьей стороны x):
   x² > 4² + 7² = 16 + 49 = 65 → x > √65 ≈ 8.06.

   С учётом целых значений: x ≥ 9.

4. **Окончательные границы:** 9 ≤ x < 11.
   Целые значения: **9, 10**.

*(Примечание: в некоторых вариантах условия учитывалось, что тупой угол может быть не напротив x. Тогда возможны x = 8, 9, 10. Уточняйте по условию конкретной задачи.)*

**Ответ:** 9, 10 (или 8, 9, 10 в зависимости от конфигурации тупого угла).

### Задача 5б (геометрия, прямоугольный треугольник)
**Условие:** В прямоугольном треугольнике один из острых углов равен 60°, а прилежащий к нему катет равен 39. Найдите гипотенузу.

**Решение:**
В прямоугольном треугольнике катет, прилежащий к углу 60°, равен половине гипотенузы, так как cos 60° = 1/2.

Гипотенуза = катет / cos 60° = 39 / (1/2) = 39 · 2 = 78.

**Ответ:** 78.

---

### Задача 5в (параллельные прямые)
**Условие:** Один из односторонних углов при параллельных прямых в 5 раз больше другого. Найдите эти углы.

**Решение:**
Сумма односторонних углов при параллельных прямых равна 180°.
Пусть меньший угол равен x, тогда больший равен 5x.
x + 5x = 180°
6x = 180°
x = 30° — меньший угол.
5x = 150° — больший угол.

**Ответ:** 30° и 150°.

## 6. Задача про Рекса (и аналогичные)

### Задача 6а (2014, I, №4)
**Условие:** Рекс загадал число, увеличил его на 20%, затем уменьшил результат на 20% и получил 48. Какое число загадал Рекс?

**Решение:**
Пусть загаданное число равно x.

1. Увеличили на 20%: x · 1.2.
2. Уменьшили на 20%: 0.8 · (x · 1.2) = 0.96x.
3. Получили 48: 0.96x = 48.
4. x = 48 / 0.96 = 4800 / 96 = 50.

**Проверка:**
- 50 увеличили на 20%: 50 · 1.2 = 60.
- 60 уменьшили на 20%: 60 · 0.8 = 48. Верно.

**Важный вывод:** Последовательное увеличение на 20% и уменьшение на 20% не возвращает к исходному числу, а даёт 0.96 от него (уменьшение на 4%).

**Ответ:** 50.

### Задача 6б (дополнительная)
**Условие:** Найдите число, если 3/7 от него равны 14.

**Решение:**
3/7 от числа = 14.
Чтобы найти всё число, делим часть на дробь:
x = 14 : (3/7) = 14 · (7/3) = 98/3 = 32 ²/₃.

**Ответ:** 32 ²/₃ (или 98/3).
