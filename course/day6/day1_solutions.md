## Задача 1. Углы треугольника (2024, I, №1д)

**а)** Пусть углы равны $4k$, $8k$, $3k$. Сумма углов треугольника $= 180^\circ$:
$$4k + 8k + 3k = 180^\circ$$
$$15k = 180^\circ$$
$$k = 12^\circ$$

Углы: $4 \cdot 12^\circ = 48^\circ$, $8 \cdot 12^\circ = 96^\circ$, $3 \cdot 12^\circ = 36^\circ$.

**б)** Один из углов ($96^\circ$) больше $90^\circ$ $\Rightarrow$ треугольник тупоугольный.

**Ответ:** $48^\circ$, $96^\circ$, $36^\circ$; тупоугольный.

## Задача 2. Смежные углы (2022, I, №1г)

Сумма смежных углов $= 180^\circ$. Пусть меньший угол $= x$, тогда больший $= x + 30^\circ$:
$$x + (x + 30^\circ) = 180^\circ$$
$$2x = 150^\circ$$
$$x = 75^\circ$$

Углы: $75^\circ$ и $75^\circ + 30^\circ = 105^\circ$.

**Ответ:** $75^\circ$ и $105^\circ$.

## Задача 3. Параллельные прямые (2023, I, №1е)

**а)** Сумма односторонних углов при параллельных прямых $= 180^\circ$.
Пусть меньший угол $= x$, тогда больший $= 5x$:
$$x + 5x = 180^\circ$$
$$6x = 180^\circ$$
$$x = 30^\circ$$
Односторонние углы: $30^\circ$ и $150^\circ$.

**б)** При пересечении двух параллельных прямых третьей образуются 8 углов. Накрест лежащие углы равны, соответственные равны. Все острые углы $= 30^\circ$, все тупые $= 150^\circ$.

**Ответ:** $30^\circ$ и $150^\circ$; $30^\circ$ — все острые, $150^\circ$ — все тупые.

**а)** $ABCD$ — параллелограмм (противоположные стороны параллельны).

**б)** В параллелограмме противоположные стороны равны:
$CD = AB = 5$, $AD = BC = 7$.

**в)** В параллелограмме сумма соседних углов $= 180^\circ$:

**Ответ:** а) параллелограмм; б) $CD = 5$, $AD = 7$; в) $120^\circ$.

Внешний угол $= 120^\circ$. Смежный с ним внутренний угол:
$$180^\circ - 120^\circ = 60^\circ$$

Один из несмежных внутренних углов $= 40^\circ$. Третий угол:
$$180^\circ - 60^\circ - 40^\circ = 80^\circ$$

**Ответ:** $60^\circ$, $40^\circ$, $80^\circ$.

## Задача 6. Равнобедренный треугольник, высота (2010, I, №6)

$\triangle ABC$, $AB = BC$. $BH \perp AC$ (высота). $2BH = AC$.

В равнобедренном треугольнике высота, проведённая к основанию, является медианой:
$$AH = HC = \frac{AC}{2} = BH$$

В $\triangle ABH$: $AH = BH$ и $\angle AHB = 90^\circ$ (так как $BH \perp AC$).
Значит $\triangle ABH$ — прямоугольный равнобедренный.
$\angle BAH = \angle ABH = 45^\circ$ (сумма острых углов прямоугольного треугольника $= 90^\circ$, углы при равных катетах равны).

$\angle BAC = \angle BAH = 45^\circ$.
В $\triangle ABC$: $AB = BC$ $\Rightarrow$ $\angle BAC = \angle BCA = 45^\circ$.
$\angle ABC = 180^\circ - 45^\circ - 45^\circ = 90^\circ$.

**Ответ:** $\angle A = \angle C = 45^\circ$, $\angle B = 90^\circ$.

## Задача 7. Неравенство треугольника (2022, I, №1е)

$AB = 5$, $BC = 1$. По неравенству треугольника:
$$|AB - BC| < AC < AB + BC$$
$$|5 - 1| < AC < 5 + 1$$
$$4 < AC < 6$$

Целые числа в этом интервале: $5$.
$AC$ — нечётное целое $\Rightarrow$ $AC = 5$.

Проверка: $5 + 1 > 5$ ✓, $5 + 5 > 1$ ✓, $1 + 5 > 5$ ✓.

**Ответ:** $5$ см.

## Задача 8. Равнобедренный треугольник, биссектриса (2010, II, №6)

$\triangle ABC$, $AB = BC$. $BL$ — биссектриса $\angle B$.

В равнобедренном треугольнике биссектриса, проведённая к основанию, является медианой и высотой. Значит:
$$AL = LC = \frac{AC}{2},\quad BL \perp AC$$

По условию $2BL = AC$, значит $BL = \frac{AC}{2} = AL = LC$.

В $\triangle ABL$: $AL = BL$ и $\angle ALB = 90^\circ$ (так как $BL \perp AC$).
Значит $\triangle ABL$ — прямоугольный равнобедренный.
$\angle BAL = \angle ABL = 45^\circ$ (сумма острых углов прямоугольного треугольника $= 90^\circ$, углы при равных катетах равны).

$\angle BAC = \angle BAL = 45^\circ$.
В $\triangle ABC$: $AB = BC$ $\Rightarrow$ $\angle BAC = \angle BCA = 45^\circ$.
$\angle ABC = 180^\circ - 45^\circ - 45^\circ = 90^\circ$.

**Ответ:** $\angle A = \angle C = 45^\circ$, $\angle B = 90^\circ$.

## Задача 10. Бобры на берегах (2023, I, №7)

**а)** Пусть $L$ — число левобережных бобров. Каждый сплясал разное целое количество танцев (минимум $1$). Сумма танцев $= 30$. Минимальная сумма при $L$ бобрах:
$$1+2+\dots+L = \frac{L(L+1)}{2} \le 30$$

$L = 7$: $28 \le 30$ ✓. $L = 8$: $36 > 30$ ✗. Наибольшее $L = 7$.

**б)** Пусть $x$ — левобережных бобров, $y$ — правобережных.
Пришло на танцы: $\frac{x}{7}$ левобережных и $\frac{y}{3}$ правобережных.
Каждый третий на танцах — с левого берега:
$$\frac{x/7}{x/7 + y/3} = \frac13$$
$$\frac{x}{7} = \frac13\left(\frac{x}{7} + \frac{y}{3}\right)$$
$$\frac{2x}{21} = \frac{y}{9}$$
$$6x = 7y \Rightarrow x = \frac{7y}{6}$$
Так как $x$ и $y$ — целые, $y$ делится на $6$, и $x > y$ (так как $\frac76 > 1$). На левом берегу больше.

**в)** Количество танцев у левобережных $= 4L$, у правобережных $= 7P$. Количество пар равно меньшему из этих чисел. Для минимального $L$ нужно равенство:
$$4L = 7P$$
$P \ge 10$, $L = \frac{7P}{4}$. Чтобы $L$ было целым, $P$ должно делиться на $4$.
$P = 12$: $L = \frac{7 \cdot 12}{4} = 21$ — минимальное $L$.

**Ответ:** а) $7$; б) на левом; в) $21$.

## Задача 12. Ваня, Даня и Таня (2007, I, №7)

У каждого по $3$ задачи. Любые двое имеют ровно $1$ общую задачу.
Суммы номеров: Таня $= 9$, Даня $= 10$, Ваня $= 14$.
Таня не решила задачу №6.

Номера задач: $1, 2, 3, 4, 5, 6$.

Для Тани ($3$ числа, сумма $9$, без $6$): возможные наборы:
$$\{1,2,6\}\text{ — есть 6, нельзя}$$
$$\{1,3,5\} = 9$$
$$\{2,3,4\} = 9$$

**Случай 1:** Таня $= \{1,3,5\}$.
Дане нужна сумма $10$, $1$ общая с Таней. Варианты:
- $\{2,3,5\}$ — имеет $2$ общих с Таней ($3$ и $5$) ✗
- $\{1,4,5\}$ — имеет $1$ общую ($5$) ✓
- $\{2,4,4\}$ — нельзя

Даня $= \{1,4,5\}$.

Ване нужна сумма $14$, $1$ общая с Таней и $1$ общая с Даней.
У Тани: $\{1,3,5\}$. У Дани: $\{1,4,5\}$.
Ваня должен взять: $1$ из $\{1,3,5\}$ (не $1$, иначе $2$ с Даней), $1$ из $\{1,4,5\}$ (не $1$), и третью.
Общие с Таней: может быть $3$ или $5$. Общие с Даней: может быть $4$ или $5$.
Если общая с Таней $= 3$, а с Даней $= 4$: Ваня имеет $3, 4, ?$ где $? = 14-3-4 = 7$ — нет задачи $7$.
Если общая с Таней $= 5$, а с Даней $= 5$: это одна и та же общая задача, но у Вани должно быть по $1$ общей с каждым — это могут быть разные задачи.
Если общая с Таней $= 5$, а с Даней $= 4$: Ваня $= \{5,4,?\}$, $? = 5$ — нельзя.
Если общая с Таней $= 3$, а с Даней $= 5$: Ваня $= \{3,5,?\}$, $? = 6$ — $3+5+6=14$ ✓.
Проверка: Ваня $\{3,5,6\}$ — общая с Таней $3$ и $5$ — это $2$ общих! ✗.

**Случай 2:** Таня $= \{2,3,4\}$.
Дане нужна сумма $10$, $1$ общая с Таней. Варианты:
- $\{1,4,5\}$ — общая $4$ с Таней ✓
- $\{2,3,5\}$ — общие $2$ и $3$ ✗

Даня $= \{1,4,5\}$.

Ване нужна сумма $14$, $1$ общая с Таней и $1$ общая с Даней.
У Тани: $\{2,3,4\}$. У Дани: $\{1,4,5\}$.
Общая с Таней: $2$, $3$ или $4$. Общая с Даней: $1$, $4$ или $5$.
Если общая с Таней $= 3$, с Даней $= 5$: Ваня $= \{3,5,?\}$, $? = 6$ — $3+5+6=14$ ✓.
Проверка: Ваня $\{3,5,6\}$ — с Таней общая $3$ (1 общая ✓), с Даней общая $5$ (1 общая ✓).

Всё сходится!

**Ответ:** Таня — $\{2,3,4\}$; Даня — $\{1,4,5\}$; Ваня — $\{3,5,6\}$.

## Задача 13. Кто стукнул Сашу (2008, I, №6)

Заявления:
1. Блип: «Сашу стукнул Влип».
2. Влип: «Сашу стукнул Глип».
3. Глип: «Меня оговорил Влип» (т.е. Влип лжёт).
4. Плип: «Я не стучал Сашу».
5. Длип: «Сашу стукнул Блип или Глип».

Трое всегда говорят правду, двое лгут.

Анализ: Если Влип прав (стукнул Глип), то Глип говорит «Влип лжёт» — но Влип прав, значит Глип лжёт. Если Глип лжёт, то Влип не лжёт — противоречия нет? Подождите, Глип говорит «меня оговорил Влип», то есть «Влип сказал неправду». Если Влип прав (стукнул Глип), то утверждение Глипа «Влип лжёт» — ложь, значит Глип лжец. Это непротиворечиво, но нужно проверить остальных.

Пусть Влип — правда (стукнул Глип). Тогда Глип — ложь (Влип не лжёт). Остаётся Блип, Плип, Длип — двое правдивы, один лжец.

Если стукнул Глип, то:
- Блип: «Влип» — ложь (стукнул Глип, не Влип) → Блип лжец.
- Плип: «Я не стучал» — правда (стукнул Глип) → Плип правдив.
- Длип: «Блип или Глип» — правда (Глип) → Длип правдив.

Итого: правдивы Влип, Плип, Длип (3). Лжецы: Блип, Глип (2). Стукнул Глип.

**Ответ:** а) Глип; б) Влип, Плип, Длип.

## Задача 14. Угадать число (2009, I, №7)

Утверждения:
1) $2x > 70$ $\Rightarrow$ $x > 35$
2) $x < 100$
3) $3x > 25$ $\Rightarrow$ $x > 8\frac13$
4) $x \ge 10$
5) $x > 5$

Три утверждения верны, два — нет.

Утверждение (3) верно для любого $x \ge 9$.
Утверждение (4) верно для $x \ge 10$.
Утверждение (5) верно для $x \ge 6$.

Переберём, какое из утверждений может быть ложным.

Если ложно (1): $x \le 35$. Остальные (2),(3),(4),(5) — $4$ верных ✗.
Если ложно (2): $x \ge 100$. Остальные (1),(3),(4),(5) — $4$ верных ✗.
Если ложно (3): $x \le 8$. Тогда (1) ложно, (4) ложно для $x<10$, (5) верно для $x\ge6$. Это $3$ ложных ✗.
Если ложно (4): $x \le 9$. Тогда (1) ложно для $x\le35$, (2) верно для $x<100$, (3) верно для $x\ge9$, (5) верно для $x\ge6$.
При $x=9$: (1) $18>70$ — нет ✗, (2) $9<100$ — да ✓, (3) $27>25$ — да ✓, (4) $9\ge10$ — нет ✗, (5) $9>5$ — да ✓.
Итого: $3$ верных (2,3,5), $2$ неверных (1,4) ✓.

Если ложно (5): $x \le 5$. Тогда (1) ложно, (2) верно, (3) ложно для $x\le5$, (4) ложно. Это $3$ ложных ✗.

**Ответ:** $9$.

## Задача 15. Рыцари и лжецы (2010, I, №7)

Первый: «Нас не больше трёх. Все мы — лжецы.»
Второй: «Нас не больше четырёх. Не все мы лжецы.»
Третий: «Нас пятеро. Трое из нас лжецы.»

Если Первый — рыцарь, то «все мы лжецы» — правда, но рыцарь не может быть лжецом. Противоречие. Значит Первый — лжец. Оба его утверждения ложны:
- «Нас не больше трёх» — ложь $\Rightarrow$ людей $> 3$.
- «Все мы лжецы» — ложь $\Rightarrow$ есть хотя бы один рыцарь.

Второй: пусть Второй — рыцарь. Тогда:
- «Нас не больше четырёх» — правда $\Rightarrow$ людей $4$ (так как $>3$ по первому условию, и $\le 4$).
- «Не все мы лжецы» — правда $\Rightarrow$ есть рыцари ✓.

Третий — лжец (так как рыцарей больше нет, а людей $4$, а не $5$):
- «Нас пятеро» — ложь ✓ (нас $4$).
- «Трое из нас лжецы» — ложь $\Rightarrow$ лжецов не $3$. При $4$ человеках: Первый — лжец, Третий — лжец, Второй — рыцарь. Лжецов $2$ (не $3$) ✓.

Проверка: Первый (лжец) — оба утверждения ложны ✓. Второй (рыцарь) — оба истинны ✓. Третий (лжец) — оба ложны ✓.

**Ответ:** $4$ человека, $2$ лжеца.

## Задача 16. Числа на бумажках (2013, I, №7)

Аня: $\{1, a\}$; Боря: $\{a, b, 3\}$; Саша: $\{2, 4, c\}$; Дима: $\{a, b, 4\}$; Егор: $\{a, b, c, d\}$.

**Условие 1:** всё, что у Ани, есть у Димы.
$\{1, a\} \subseteq \{a, b, 4\}$ $\Rightarrow$ $1$ есть у Димы. Так как у Димы числа $a, b, 4$, то либо $b = 1$, либо $a = 1$.
Если $a = 1$, то у Ани $\{1, 1\}$ — но числа не повторяются, значит $a \ne 1$.
Следовательно, $b = 1$.

**Условие 2:** всё, что у Бори, Саши и Димы, есть у Егора.
Боря: $\{a, 1, 3\}$; Саша: $\{2, 4, c\}$; Дима: $\{a, 1, 4\}$.
Объединение: $\{a, 1, 3, 2, 4, c\}$.
Это подмножество Егора $\{a, 1, c, d\}$.

Значит числа $2, 3, 4$ должны быть среди $\{a, 1, c, d\}$.
Так как $1$ уже есть, $a, c, d$ покрывают $\{2, 3, 4\}$.

$a$ не может быть $1$ (уже у Ани, и это разные числа). $a$ может быть $2, 3$ или $4$.
$c$ может быть $2, 3$ или $4$.

Подберём: пусть $a = 2$. Тогда в объединении $\{2, 1, 3, 2, 4, c\} = \{1, 2, 3, 4, c\}$.
Для Егора: $\{2, 1, c, d\}$ должны содержать $\{1, 2, 3, 4, c\}$.
$c$ должен быть $3$ или $4$. Если $c = 3$, то у Егора $\{2, 1, 3, d\}$, нужно покрыть $\{1, 2, 3, 4\}$ $\Rightarrow$ $d = 4$.
Проверка: у Ани $\{1, 2\}$ $\subseteq$ у Димы $\{2, 1, 4\}$ ✓.
У Бори $\{2, 1, 3\}$, у Саши $\{2, 4, 3\}$, у Димы $\{2, 1, 4\}$ — всё есть у Егора $\{2, 1, 3, 4\}$ ✓.

**Ответ:** $a = 2$, $b = 1$, $c = 3$, $d = 4$.
