## Задача 1. Бег по стадиону (2025, I, №5)

$v_K = 2$ км/ч $= \frac{2000}{60} = \frac{100}{3}$ м/мин. $v_E = 3v_K = 100$ м/мин.

План: встретиться на противоположной стороне — каждый пробегает $L/2$ метров.
Плановое время Катерины: $t_K^{план} = \frac{L/2}{100/3} = \frac{3L}{200}$ мин.
Плановое время Эрвина: $t_E^{план} = \frac{L/2}{100} = \frac{L}{200}$ мин.
Разница: $\frac{3L}{200} - \frac{L}{200} = \frac{L}{100} = 3$ мин $\Rightarrow$ $L_{план} = 300$ м.

Факт: Эрвин пробежал $\frac{L}{2} + 50$ м. Его время: $t_E = \frac{L/2 + 50}{100}$ мин.
Катерина бежала на $3$ минуты дольше: $t_K = t_E + 3$ мин.
Сумма расстояний, пройденных ими, равна $L$:
$$\frac{100}{3} \cdot (t_E + 3) + \left(\frac{L}{2} + 50\right) = L$$

Подставляем $t_E = \frac{L/2 + 50}{100} = \frac{L + 100}{200}$:
$$\frac{100}{3}\left(\frac{L + 100}{200} + 3\right) + \frac{L}{2} + 50 = L$$
$$\frac{L + 100}{6} + 100 + \frac{L}{2} + 50 = L$$
$$\frac{L + 100}{6} + \frac{L}{2} + 150 = L$$

Умножаем на $6$:
$$L + 100 + 3L + 900 = 6L$$
$$4L + 1000 = 6L$$
$$2L = 1000$$
$$L = 500$$

**Ответ:** $500$ м.

## Задача 2. Эрвин ведёт Катерину (2023, I, №5)

$v_1 = 2{,}5$ км/ч $= 2500$ м/ч. После остановки: $v_2 = 2500 + 500 = 3000$ м/ч $= 3$ км/ч.
Задержка $= 4$ мин $= \frac{1}{15}$ ч.

Пусть $S$ (км) — расстояние от садика до дома.

Плановое время: $T_{план} = \frac{S}{2{,}5} = \frac{2S}{5}$ ч.

Фактическое время:
$$t_1 = \frac{S/9}{2{,}5} = \frac{S}{22{,}5} = \frac{2S}{45}\text{ ч}$$
$$t_{задерж} = \frac{1}{15}\text{ ч}$$
$$t_2 = \frac{8S/9}{3} = \frac{8S}{27}\text{ ч}$$
$$T_{факт} = \frac{2S}{45} + \frac{1}{15} + \frac{8S}{27}$$

$T_{факт} = T_{план}$:
$$\frac{2S}{45} + \frac{1}{15} + \frac{8S}{27} = \frac{2S}{5}$$

Общий знаменатель $135$:
$$\frac{6S}{135} + \frac{9}{135} + \frac{40S}{135} = \frac{54S}{135}$$
$$46S + 9 = 54S$$
$$8S = 9$$
$$S = \frac{9}{8} = 1{,}125\text{ км} = 1125\text{ м}$$

Средняя скорость: $v_{ср} = \frac{S}{T_{план}} = \frac{1{,}125}{1{,}125/2{,}5} = 2{,}5$ км/ч (равна начальной скорости, так как пришли вовремя).

**Ответ:** $1{,}125$ км ($1125$ м), $v_{ср} = 2{,}5$ км/ч.

## Задача 3. Эрвин из школы (2021, I, №5)

$v_1 = 4$ км/ч. $v_2 = 150$ м/мин $= \frac{150 \cdot 60}{1000} = 9$ км/ч.
$\Delta t = 15$ мин $= \frac{1}{4}$ ч.

Пусть $S$ (км) — путь. Плановое время: $T_{план} = \frac{S}{4}$.

Фактическое время:
$$t_1 = \frac{S/3}{4} = \frac{S}{12}\text{ ч}$$
$$t_2 = \frac{2S/3}{9} = \frac{2S}{27}\text{ ч}$$

Разница: $T_{план} - T_{факт} = \frac14$:
$$\frac{S}{4} - \left(\frac{S}{12} + \frac{2S}{27}\right) = \frac14$$
$$\frac{S}{4} - \frac{S}{12} - \frac{2S}{27} = \frac14$$

Общий знаменатель $108$:
$$\frac{27S - 9S - 8S}{108} = \frac14$$
$$\frac{10S}{108} = \frac14$$
$$10S = 27$$
$$S = 2{,}7\text{ км}$$

**Ответ:** $2{,}7$ км.

## Задача 4. Эрвин в садике (2014, I, №5)

Расстояние до садика $3$ км. Эрвин идёт туда со скоростью $5$ км/ч. Катерина идёт из садика домой со скоростью $4$ км/ч. Они выходят одновременно навстречу друг другу.

Пусть $x$ км прошёл Эрвин до встречи. Тогда Катерина прошла $3 - x$ км.
Время одинаковое:
$$\frac{x}{5} = \frac{3 - x}{4}$$
$$4x = 5(3 - x)$$
$$4x = 15 - 5x$$
$$9x = 15$$
$$x = \frac{15}{9} = \frac{5}{3}\text{ км}$$

Время встречи:
$$t = \frac{x}{5} = \frac{5/3}{5} = \frac{1}{3}\text{ ч} = 20\text{ мин}$$

**Ответ:** $20$ мин.

## Задача 5. Вороны на деревьях (2006, II тур, №1)

Всего $35$ ворон. Пусть $x$ ворон на берёзе, $y$ — на ольхе.
$$x + y = 35$$

После того как с берёзы улетело $5$ ворон:
$$x - 5 = 2y$$

Из второго уравнения: $x = 2y + 5$.
Подставляем в первое:
$$2y + 5 + y = 35$$
$$3y = 30$$
$$y = 10,\quad x = 2 \cdot 10 + 5 = 25$$

**Ответ:** на берёзе $25$, на ольхе $10$.

## Задача 6. Рекс и число (2007, I, №3)

Пусть $x$ — задуманное число. По заданию: $4x + 15$. По факту (перепутал): $15x + 4$.
Результаты равны:
$$4x + 15 = 15x + 4$$
$$15 - 4 = 15x - 4x$$
$$11 = 11x$$
$$x = 1$$

**Ответ:** $1$.

## Задача 7. Рекс вылупился (2008, I, №3)

Рексу сейчас $13$ дней, Саше $3$ дня. Через $t$ дней Рекс станет втрое старше Саши:
$$13 + t = 3(3 + t)$$
$$13 + t = 9 + 3t$$
$$13 - 9 = 3t - t$$
$$4 = 2t$$
$$t = 2$$

**Ответ:** через $2$ дня.

## Задача 8. Сон Эрвина (2013, I, №5)

Сутки $= 24$ ч. Пусть $d$ — сон днём (когда спит), $n$ — сон ночью.
Сумма сна $= d + n$. Сумма бодрствования $= 24 - (d + n)$.
По условию $d + n = 24 - (d + n)$ $\Rightarrow$ $2(d + n) = 24$ $\Rightarrow$ $d + n = 12$ ч.

Если бы Эрвин не спал днём, то спал бы ночью $n'$ часов. По условию, когда он спит днём, то ночью спит в $1{,}2 = \frac65$ раза меньше: $n = \frac{5}{6}n'$.

Если бы не спал днём: $d = 0$, тогда $n' = 12$ ч (вся сумма сна приходится на ночь).
Сейчас: $n = \frac{5}{6} \cdot 12 = 10$ ч.
$$d = 12 - n = 12 - 10 = 2\text{ ч}$$

**Ответ:** $2$ часа.

## Задача 9. Lego детальки (2024, I, №5)

Первый набор: $N_1 = 1000$ деталек. Прямоугольных $= 38\% = 0{,}38 \cdot 1000 = 380$.
Квадратных $= 1000 - 380 = 620$.

Второй набор: квадратных $640$ — это $80\%$ (так как $20\%$ прямоугольных).
$$N_2 = \frac{640}{0{,}8} = 800$$
Прямоугольных $= 0{,}2 \cdot 800 = 160$.

Общий набор: $N = 1000 + 800 = 1800$.
Прямоугольных всего: $380 + 160 = 540$.
Процент прямоугольных: $\frac{540}{1800} \cdot 100\% = 30\%$.

**Ответ:** $30\%$.

## Задача 10. Умпа и Лумпа (2021, I, №4)

Пусть $x$ грибов собрал Умпа, $y$ — Лумпа.
$$x + y = 130$$

После потери: у Умпы осталось $75\% = 0{,}75x$, у Лумпы $80\% = 0{,}8y$.
Утром у Умпы на $50\%$ больше, чем у Лумпы:
$$0{,}75x = 1{,}5 \cdot 0{,}8y = 1{,}2y$$

Из второго: $x = \frac{1{,}2}{0{,}75}y = 1{,}6y$.
Подставляем в первое:
$$1{,}6y + y = 130$$
$$2{,}6y = 130$$
$$y = 50,\quad x = 1{,}6 \cdot 50 = 80$$

**Ответ:** Умпа — $80$, Лумпа — $50$.

## Задача 11. Эрвин строит башню (2012, I, №6)

Эрвин строит башню за $30$ мин. Скорость: $\frac{1}{30}$ башни/мин.
Саня в $3$ раза быстрее: $t_С = \frac{30}{3} = 10$ мин, скорость $\frac{1}{10}$ башни/мин.
Егор в $1{,}5$ раза дольше Сани: $t_Е = 1{,}5 \cdot 10 = 15$ мин, скорость $\frac{1}{15}$ башни/мин.

Общая скорость:
$$\frac{1}{30} + \frac{1}{10} + \frac{1}{15} = \frac{1 + 3 + 2}{30} = \frac{6}{30} = \frac{1}{5}$$

Время: $1 : \frac{1}{5} = 5$ мин.

**Ответ:** $5$ мин.

Эрвин съедает за $9$ мин. Скорость: $\frac{1}{9}$ тарелки/мин.

Время: $1 : \frac{1}{3} = 3$ мин.

## Задача 14. Эрвин набирает вес (2010, I, №4)

Пусть $x$ (г) — вес при рождении.

После $1$-го месяца: $x \cdot 1{,}25 = 1{,}25x$.
После $2$-го месяца: $1{,}25x \cdot 1{,}3 = 1{,}625x$.
После $3$-го месяца: $1{,}625x + 1170 = 6240$.

$$1{,}625x = 6240 - 1170 = 5070$$
$$x = \frac{5070}{1{,}625} = 3120\text{ г}$$

Прибавка за $3$ месяца:
$$\frac{6240 - 3120}{3120} \cdot 100\% = \frac{3120}{3120} \cdot 100\% = 100\%$$

**Ответ:** а) $3120$ г; б) $100\%$.

## Задача 15. Уровень компота (2007, I, №6)

Пусть начальный уровень компота $= 1$.

После съедания половины персиков уровень понизился на $\frac13$:
новый уровень $= 1 - \frac13 = \frac23$.

Половина персиков понизила уровень на $\frac13$. Значит, оставшаяся половина персиков при съедании тоже понизит уровень на $\frac13 \cdot \frac12 = \frac16$ от начального уровня.

Новый уровень после съедания половины оставшихся:
$$\frac23 - \frac16 = \frac{4-1}{6} = \frac36 = \frac12$$

Падение от полученного уровня (от $\frac23$):
$$\frac{1/6}{2/3} = \frac16 \cdot \frac32 = \frac{3}{12} = \frac14$$

**Ответ:** на $\frac14$ ($25\%$).

## Задача 16. Башенки из кубиков (2022, I, №5)

Скорость сборки Катерины: $\frac{12}{5} = 2{,}4$ куб/мин.
Время на башню из $5$ кубиков: $\frac{5}{2{,}4} = \frac{25}{12}$ мин.

За это время Эрвин строит $2 \cdot 9 = 18$ кубиков.
Скорость Эрвина: $\frac{18}{25/12} = 18 \cdot \frac{12}{25} = \frac{216}{25} = 8{,}64$ куб/мин.

Катерина мешает: разбирает со скоростью $2{,}4$ куб/мин.
Чистая скорость сборки (Эрвин строит, Катерина разбирает):
$$\frac{216}{25} - \frac{12}{5} = \frac{216}{25} - \frac{60}{25} = \frac{156}{25}\text{ куб/мин}$$

Время на $26$ кубиков:
$$t = \frac{26}{156/25} = 26 \cdot \frac{25}{156} = \frac{650}{156} = \frac{25}{6} = 4\frac16\text{ мин} = 4\text{ мин }10\text{ сек}$$

**Ответ:** $4$ мин $10$ сек.
